#17 - เตรียมสอบ Google Cloud Digital Leader

2024-02-16

For English version, readers can follow here on Medium.

สวัสดีครับ หลังจากที่ไม่ได้เขียนบล็อกไปสัปดาห์นึงไปก็ไม่ใช่อะไร ตอนนั้นเราไปเตรียมสอบ Google Cloud Digital Leader มาเมื่อสัปดาห์ที่แล้ว และได้สอบมาเมื่อวานนี้เอง

แต่ก่อนอื่น เราไม่เสียเวลาเกริ่นนาน เรามาเริ่มพูดถึงการเตรียมตัวสอบดีกว่าครับ

Google Cloud Digital Leader

Google Cloud Digital Leader [1] เป็นการสอบ Google Cloud Certificate ที่วัดความรู้พื้นฐานเรื่องคลาวด์ ผลิตภัณฑ์ Google Cloud และการนำไปใช้เพื่อบรรลุเป้าหมายทางธุรกิจ

คลาวด์ (หรือเรียกว่า Cloud Computing) คือบริการที่ผู้ใช้งานสามารถเข้าถึงเซิร์ฟเวอร์ที่เชื่อมต่อกันบนเครือข่าย และทรัพยากรผ่านทางอินเตอร์เน็ต แทนที่จะใช้คอมพิวเตอร์ส่วนบุคคล หรือเซิร์ฟเวอร์ที่ตั้งอยู่ใน local (On-Premise)

จุดนี้ทำให้การใช้คลาวด์มีข้อดีคือเราไม่ต้องซื้อเซิร์ฟเวอร์มาติดตั้งเอาเอง และไม่ต้องลงทุนกับการติดตั้ง Infrastructure, ระบบความปลอดภัย, ระบบไฟฟ้า, ระบบเครือข่าย และอื่น ๆ

การสอบใบ Certificate นี้มีข้อดีคือได้ทั้ง Knowledge และ Skill ที่สามารถต่อยอดไปยังสายงานเรามากกว่า โดยการสอบใบ Certificate นี้ทำให้มีแรงจูงใจในการได้ความรู้ และทดสอบว่าเราผ่านมาตราฐานนั้นหรือไม่ แล้วเอาความรู้ที่ใช้ไปประยุกต์ต่อไป โดยการสอบ Certificate ของ Google Cloud Platform (GCP) แบ่งเป็น 3 ระดับ ได้แก่

  • Foundational เป็นการสอบความรู้พื้นฐานที่เกี่ยวกับคลาวด์ ผลิตภัณฑ์กับเครื่องมือของ Google Cloud รวมถึง Use case ที่เกี่ยวข้อง
  • Associate เป็นการวัดความรู้พื้นฐานของการ Deploy และการดูแล Cloud Project
  • Professional เป็นการวัดความรู้ในสายงานที่เกี่ยวข้องกับแต่ละสายงาน

เราสามารถแสดงแผนภาพของการทดสอบ Google Cloud Certificate ได้ตามด้านล่างนี้ โดยเราไม่จำเป็นต้องสอบเรียงตามระดับขึ้นไปตั้งแต่ Foundational, Associate และ Professional

การสอบ Cloud Digital Leader เป็นการสอบ Certificate แบบ Foundational ที่เทียบเท่ากับการทดสอบ Certificate ตัวเริ่มต้นของ Microsoft และ AWS ได้แก่ Azure Fundamental และ AWS Cloud Practitioner

เนื้อหาที่ออกข้อสอบ

เนื้อหาสำหรับการเตรียมตัวสอบระบบเก่า (non-beta) ที่เราสอบไปมีเนื้อหาทั้งหมด 4 Topic [1] โดยแสดงตามด้านล่างนี้

  • Digital Transformation with Google Cloud (~ 10% of the exam)
  • Innovating with data and Google Cloud (~ 30% of the exam)
  • Infrastructure and application modernization (~ 30% of the exam)
  • Understanding Google Cloud security and operations (~ 30% of the exam)

ส่วนระบบใหม่ (beta) [1] ที่จะเปลี่ยนก็จะมีเนื้อหาที่เตรียมสอบทั้งหมด 6 Topic ที่แสดงตามด้านล่างนี้

  • Digital Transformation with Google Cloud (~17% of the exam)
  • Exploring Data Transformation with Google Cloud (~17% of the exam)
  • Innovating with Google Cloud Artificial Intelligence (~17% of the exam)
  • Modernize Infrastructure and Applications with Google Cloud (~17% of the exam)
  • Trust and Security with Google Cloud (~17% of the exam)
  • Scaling with Google Cloud Operations (~17% of the exam)

การเตรียมตัวสอบ

เอาล่ะ เรามาพูดแหล่งเรียนรู้สำหรับการเตรียมตัวสอบ

ปกติบริการคลาวด์ในแต่ละเจ้าจะมีหลักสูตรสำหรับการเรียนได้ฟรีเพื่อที่จะเรียนรู้เทคโนโลยีต่าง ๆ ของคลาวด์แต่ละเจ้าไปด้วย เพื่อที่จะให้ทางนั้นขายบริการได้นั่นเอง โดยของ Google Cloud Digital Leader นี้เราสามารถเรียนได้ใน Google Cloud Skill Boost

วิธีการเรียนของเราคือจดลงสมุด (หรือจดลงไอแพดก็ได้) โดยเรียนตามในคอร์สนั้นให้จบรอบแรกก่อน เมื่อเจอจุดไหนที่ไม่เข้าใจเราก็เปิดคลิปวิดีโอทวนซ้ำเพื่อให้เราเข้าใจ พอเราเรียนจบในแต่ละบทแล้วมันจะมีสอบเพื่อทบทวนความรู้ที่เราเรียนมาครับ

นอกจากการเรียนในคอร์สนั้น ๆ แล้ว แนะนำให้ลองเล่นบริการของ Google Cloud ตามที่ในคอร์สนั้นแนะนำ หรือถ้าไม่สะดวกก็อ่านเอกสารที่มีมาให้ในเว็บของ Google Cloud เอง ในนั้นอธิบายได้ละเอียดพอที่จะนำไปสอบได้ครับ

อีกอย่าง กรณีที่เรียนแล้วต้องการความรู้เพิ่มเติม เราก็เรียนคอร์สที่สอนฟรีในยูทูป (ตามด้านล่างนี้) ได้เลย โดยคลิปนี้อยู่ใน Channel FreeCodeCamp.org

เมื่อเรียนมาหมดแล้ว เราก็ลองทำข้อสอบดู โดยผู้อ่านสามารถหาได้ใน Google ครับ

ส่วนของเรา (ที่ข้อสอบยังเป็นแบบเก่า) เราซื้อข้อสอบเก่าจากหน้าเว็บ P2Pexams ที่เป็นไฟล์ PDF มาสอบ ในนั้นจะมีข้อสอบให้ทำราว ๆ 300 กว่าข้อ นอกจากแบบทดสอบแล้ว ตัวเฉลยยังมีการอธิบายเหตุผลว่าทำไมต้องตอบ และมีลิ้งค์ให้ไปอ่านเพิ่มได้อีกด้วย (ส่วนนี้ผู้อ่านไม่จำเป็นต้องซื้อก็ได้)

การสมัครสอบ

ต่อมาเมื่อเตรียมพร้อมแล้ว เราก็นัดสอบผ่านหน้าเว็บของ Google Cloud เองที่ใช้บริการของเว็บ Webassessor โดยการกดปุ่ม Register จากนั้นเลือกได้ว่าจะสอบแบบ On-site หรือ Online ที่การสอบ Online มันจะเตรียมตัวยุ่งยากกว่า On-site หน่อย (ไม่หน่อยอ่ะ) ตามในบล็อก Medium นี้ ที่เราจะต้องเคลียร์ห้อง เคลียร์โต๊ะ แถมระหว่างการสอบห้ามมีเสียงดัง และห้ามคนเดินผ่านอีกก [2]

ส่วนการสอบแบบ On-site ที่เราเลือกสอบไปนั้นดูสบายกว่าการสอบแบบ Online ที่สถานที่สอบนั้นเตรียมอุปกรณ์มาให้เรียบร้อยแล้ว เราก็เพียงแค่เลือกสถานที่สอบ เลือกวันและเวลาสอบ จากนั้นจ่ายเงิน โดยค่าสมัครสอบแบบเก่าคิดเงินพันกว่าบาท (ประมาณ 55 USD)

เมื่อจ่ายเงินเสร็จแล้วจะมีเมลมาแจ้งเกี่ยวกับการสอบโดยให้เราปริ้นท์เมลนั้นไปให้ทางพนักงานที่สถานที่สอบเพื่อเข้าสอบตามวันและเวลาที่ระบุ

ของเราตอนที่เราไปสอบนั้น เราสอบที่บริษัท The Enterprise Resources Training (ที่อยู่ในอาคารชาญอิสระ 2 โดยดูได้จากลิ้งค์ Google Map) เมื่อวานนี้ตอนบ่ายไป

ประสบการณ์การสอบ

การเตรียมตัวสอบ เราเตรียมตัวตามที่เราเขียนไปแล้วในหัวข้อก่อนหน้า โดยใช้เวลา 2-3 อาทิตย์พอ ๆ กับคนในบล็อก Medium ข้างบนนั้น เมื่อถึงวันสอบ เราก็เข้าไปสอบตามสถานที่ที่ระบุไป

เมื่อไปถึงสถานที่สอบ เรายื่นอีเมลที่ปริ้นท์มาที่ระบุชื่อ นามสกุล รหัสการสอบแก่พนักงาน จากนั้นรอพนักงานเตรียมสถานที่เรียบร้อย และเมื่อถึงเวลาเริ่มสอบ เราก็เข้าไปสอบ โดยเวลาที่มีให้ในการสอบมีทั้งหมด 90 นาที ปรากฏว่าทำข้อสอบ และรีวิวคำตอบไปประมาณ 30 กว่านาที จากนั้นกดส่ง Submit ไป

ผลการสอบที่ได้พบว่า สอบผ่าน

เมื่อเรา สอบผ่าน แล้วไม่กี่ชั่วโมง ตัวใบเซอร์ก็ส่งมาทางอีเมลเรียบร้อยฮะ โดยเป็นใบเซอร์ตามภาพด้านล่างนี้ครับ

ตัวใบเซอร์นี้มีอายุ 3 ปีครับ เมื่อใกล้จะหมดอายุก็ต้องมาสอบเพื่อต่ออายุฮะ นอกจากนี้ใบเซอร์ตัวต่อไปของเราก็เป็นใบเซอร์ Data Engineer นี่เกี่ยวข้องกับงานที่เรากำลังมองหาอยู่นะแหละ

สรุป

สำหรับคนที่สนใจ และต้องการ Knowledge และ Skill พื้นฐานสำหรับคลาวด์ และบริการบน Google Cloud แล้ว เราแนะนำให้เตรียมตัวตามข้างบนนี้ และสอบ Google Certificate Cloud Digital Leader ครับ
