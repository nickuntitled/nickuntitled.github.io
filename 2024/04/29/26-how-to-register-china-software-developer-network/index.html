#26 - วิธีสมัครบัญชีคอมมู Developer/Data/AI จีนอย่าง CSDN

2024-04-29

CSDN (ที่ย่อมาจาก Chinese Software Developer Network หรือ China Software Developer Network) เป็น Community สำหรับนักพัฒนาซอฟต์แวร์ กับคนที่ทำงานทางด้าน Data และ AI ที่ใหญ่ที่สุดแห่งหนึ่งในประเทศจีน

เว็บไซต์นี้ให้บริการ

  • เว็บบอร์ด ที่จะมีนักพัฒนาซอฟต์แวร์มาโพสกระทู้ในนั้นเพื่อสอบถามปัญหาแล้วมีคนมาตอบ เนื้อหาในนั้นส่วนใหญ่จะเป็นเรื่องการเขียนโปรแกรมเป็นหลัก
  • เว็บบล็อกสำหรับนักพัฒนา
  • ข่าวสาร IT กับการหางาน
  • เผยแพร่ Source Code (แนวเดียวกันกับ GitHub)
  • และให้บริการคอร์สเรียนออนไลน์

บริการทั้งหมดที่กล่าวมาตามข้างบนนั้นเขียนด้วยภาษาจีนตัวย่อ ส่วนภาษาจีนตัวเต็มกับภาษาอังกฤษ เราจะพบได้บ้าง แต่พบได้ไม่บ่อยครับ

ส่วนเนื้อหาในนั้นมีหลายหัวข้อมากที่น่าสนใจ ตั้งแต่

  • Front-end Development (前端)
  • Back-end Development (后端)
  • Mobile Development (移动开发)
  • Java
  • Python
  • Artificial Intelligence (人工智能)
  • AI Generative Content
  • Big Data (大数据)
  • Database (数据库)
  • Data Structure and Algorithms (数据结构与算法)
  • Cloud Platform (云平台)
  • Software Engineering (软件工程)
  • User Experience Design (用户体验设计)
  • Game (游戏)
  • Math (数学)
  • เป็นต้น

เนื้อหาเหล่านี้ ก็เหมาะกับนักพัฒนาที่รู้ภาษาจีนที่สนใจอ่านเนื้อหาที่เกี่ยวข้องกับการเขียนโปรแกรมในนั้น

อย่างไรก็ดี การสมัครใช้งานเว็บไซต์นี้จำเป็นต้องมีบัญชี WeChat (微信) หรือเบอร์โทรศัพท์ที่ขึ้นต้นด้วยเบอร์ตามที่กำหนด ได้แก่

  • +1 สหรัฐอเมริกา และแคนาดา
  • +86 จีน
  • +852 อ่องกง
  • +886 ไต้หวัน
  • และอื่น ๆ

โดยเมื่อดูในรายการนั้นก็พบว่าไม่มีประเทศไทย (ที่ใช้ +66) แต่ผู้อ่านไม่ต้องเสียใจไป ในบทความนี้เราจะแนะนำวิธีการสมัครเว็บนี้

แต่ก่อนอื่น เราจำเป็นต้องมีเบอร์จีนเสียก่อน

การสมัครเบอร์จีน

การสมัครเพื่อรับเบอร์จีนนั้น โดยปกติเราจำเป็นต้องไปที่ประเทศจีนเพื่อขอเบอร์โทรศัพท์ อย่างไรก็ดี ในไทยมีเจ้าหนึ่งที่ให้บริการอยู่คือ CMLink ที่ให้บริการโดย China Mobile ที่ใช้สัญญาณไทยผ่านผู้ให้บริการ True

ในบทความนี้เราจะมาแนะนำการซื้อซิม การสมัครใช้บริการเพื่อเชื่อมเบอร์จีนกันเสียก่อนที่ะไปสมัครใช้งานในเว็บไซต์ CSDN

การซื้อซิม

การซื้อซิม CMLink ทำได้โดยเข้าเว็บไซต์ของ CMLink เสียก่อน เมื่อเข้ามาหน้าเว็บไซต์แล้ว ให้ผู้อ่านเลือกที่สั่งซื้อซิม

จากนั้น ระบบจะเข้ามาหน้าจอตามภาพด้านล่าง ผู้อ่านสามารถกดเลือก

  • ต้องการเบอร์ใหม่ กรณีที่ต้องการเบอร์โทรศัพท์ใหม่ไปเลย
  • หรือย้ายค่ายเบอร์เดิม กรณีที่ต้องการย้ายค่ายจากเจ้าอื่น

เมื่อเลือกเสร็จแล้ว ให้ผู้อ่านเลือกแพคเกจตามที่ต้องการ ตัว CMLink จะมีแพคเกจให้เลือกเป็นรายเดือน หรือรายปีก็ได้

เมื่อเลือกเสร็จแล้ว กดยืนยันคำสั่งซื้อ จากนั้นระบบจะปรากฏหน้าจอให้กรอกข้อมูลส่วนตัว พร้อมกับอัพโหลดบัตรประชาชน/พาสพอร์ต ตามภาพด้านล่างนี้

เมื่อกรอกข้อมูลเรียบร้อยแล้ว ให้กดยืนยันคำสั่งซื้ออีกรอบ ระบบจะปรากฏหน้าจอเพื่อให้ผู้อ่านจ่ายเงิน ส่วนนี้ให้ผู้อ่านทำตามหน้าจอเพื่อจ่ายเงินให้เรียบร้อย

หลังจากจ่ายเงินเสร็จแล้ว ระบบจะตรวจสอบข้อมูล กรณีที่ข้อมูลถูกต้อง ทางบริษัทจะส่งซิมผ่านผู้ให้บริการ Best Express ที่ใช้ระยะเวลาประมาณสองสามวัน ตัวซิมก็มาถึงบ้าน

เมื่อผู้อ่านได้ซิมมาเรียบร้อย ให้ใส่ซิมกับโทรศัพท์/แท็บเล็ตที่ต้องการ จากนั้นตัวซิมจะได้รับการเปิดใช้งาน

การเชื่อมซิมกับเบอร์โทรศัพท์ที่จีน

เมื่อเปิดใช้งานแล้ว ให้เข้าไปที่หน้าล็อคอินของตัว CMLink ใส่เบอร์โทรศัพท์ของซิมที่ผู้อ่านซื้อมา แล้วกดรับ OTP ตัวโค้ดจะถูกส่งมายังโทรศัพท์/แท็บเล็ตนั้น ๆ

ให้ผู้อ่านกรอกเลขที่ได้ลงไปในช่องใต้ช่องที่ใส่เบอร์โทรศัพท์ จากนั้นกดปุ่มเข้าสู่ระบบ

เมื่อเข้าสู่ระบบแล้ว ระบบจะปรากฏหน้าจอตามด้านล่างนี้ ให้กดปุ่มบริการเชื่อมเบอร์จีนที่หน้าจอด้านซ้าย

จากนั้น ระบบจะเลื่อนหน้าจอลงมา โดยจะมีเบอร์จีนกับเบอร์ฮ่องกงให้เลือก ส่วนนี้ให้เลือกเป็นเบอร์จีน จากนั้นกดปุ่มขอเบอร์ China Mobile จีนใหม่

เมื่อกดปุ่ม แล้วให้ผู้อ่านกรอกรายละเอียด พร้อมกับอัพโหลดภาพพาสพอร์ต และวิดีโอที่เป็นหน้าผู้อ่านพร้อมกับถือพาสพอร์ตลงไปในระบบ แล้วก็สมัครใช้บริการ

ส่วนนี้ผู้อ่านรอไม่กี่วัน ระบบก็จะแจ้งว่าได้เปิดการใช้บริการเบอร์จีนเรียบร้อย โดยผู้อ่านสามารถล็อคอินแล้วเข้ามาดูตรงส่วนบริการเชื่อมเบอร์จีนได้ครับ

เมื่อผู้ใช้มีเบอร์จีนแล้ว ผู้อ่านสามารถใช้รับส่ง SMS รวมถึงรับโอนสายเรียกเข้ามาที่เบอร์ไทยได้อีก ส่วนนี้ถือว่าดีมาก

ต่อมา เรามาพูดถึงการสมัครใช้บริการเว็บ CSDN

การสมัครใช้งานเว็บ CSDN

การสมัครใช้บริการเว็บนี้ทำได้โดยเข้ามาที่หน้าเว็บหลักเสียก่อน (ตามภาพ) จากนั้นให้กดทีปุ่ม 登录 (หรือเข้าสู่ระบบ) ที่มุมบนขวาของหน้าเว็บไซต์

เมื่อกดปุ่มเข้าสู่ระบบที่มุมบนขวาแล้ว ระบบจะปรากฏหน้าจอเข้าสู่ระบบที่อนุญาตให้ผู้ใช้ใช้ WeChat หรือใช้เบอร์โทรศัพท์ตามประเทศที่ระบุไว้ในนั้นได้ครับ

ในบทความนี้เราจะใช้เบอร์จีนที่เปิดผ่าน CMLink มาใช้ ทำได้โดยการพิมพ์เบอร์โทรศัพท์แล้วกดปุ่ม 获取验证码 (รับรหัสยืนยัน)

หลังจากกดปุ่มรับรหัสยืนยันแล้ว ระบบจะส่งข้อความมาทาง SMS ให้เราพิมพ์ตัวเลขลงไปในกล่องข้อความใต้เบอร์โทร จากนั้นติ๊กถูกที่เครื่องหมายถูกด้านล่าง และกดปุ่ม 登录/注册 (เข้าสู่ระบบ/สมัครสมาชิก) เพื่อเข้าสู่ระบบ

กรณีที่ผู้ใช้ยังไม่มีบัญชี ระบบจะเข้ามาหน้าสมัครสมาชิก ให้ผู้ใช้กรอกรายละเอียดแล้วสมัครสมาชิกตามปกติครับ แต่ถ้ามีสมาชิกอยู่แล้ว ระบบจะเปลี่ยนหน้ามายังหน้าหลักของเว็บไซต์

แค่นี้ ผู้ใช้ก็ใช้งานเว็บไซต์ CSDN ได้แล้ว

สรุป

เว็บ CSDN (Chinese Software Developer Network) เป็น Community สำหรับนักพัฒนาซอฟต์แวร์ที่ใหญ่ที่สุดแห่งหนึ่งในประเทศจีน ที่อนุญาตให้ผู้ใช้ใช้งานเว็บบอร์ด กับบล็อก ร่วมกับดูข่าวสาร IT กับการหางาน และมีคอร์สให้เรียน

อย่างไรก็ดี เว็บไซต์นี้อนุญาตให้ผู้ใช้เพียงไม่กี่ประเทศสามารถสมัครเข้าไปใช้งานได้ ในบทความนี้เราแนะนำการสมัครเบอร์จีนผ่าน CMLink เพื่อสมัครใช้งานเว็บไซต์นี้ครับ

ส่งท้าย

สำหรับผู้อ่านที่เห็นว่าบทความนี้ดี มีประโยชน์ ให้กดไลค์ หรือกดแชร์ไปยังแพลตฟอร์มโซเชียลต่าง ๆ นอกจากนี้ ผู้อ่านยังติดตามได้่ใน Linkedin หรือ X (หรือ Twitter) ได้ครับ
