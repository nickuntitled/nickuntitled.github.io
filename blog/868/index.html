เมื่อ Microsoft Office for Android Tablet มาช้าเกินไป

Nov 6, 2014

 Tweet

ปกติโปรแกรมอย่าง Microsoft Office ที่เราใช้งานกันอยู่ในปัจจุบันนี้ เดิมทีมีให้ใช้งานได้บนระบบปฏิบัติการ Microsoft Windows กับระบบปฏิบัติการ Mac OS X เท่านั้น ต่อมาในปีนี้ที่ผ่านมาทางบริษัท Microsoft ได้ปล่อยโปรแกรมอย่าง Microsoft Office ที่ประกอบไปด้วย Microsoft Word, Excel และ PowerPoint ลงบนอุปกรณ์อย่าง iPad

จุดนี้เองเราคิดว่าเป็นสัญญาณที่ดีที่เราจะได้ลดการใช้งานระบบปฏิบัติการ Windows ลงแล้วเปลี่ยนมาใช้งานจาก tablet มากขึ้นเนื่องจากแบตเตอรี่ที่ยาวนานกว่าโน้ตบุ๊คหลายรุ่นที่มีประสิทธิภาพสูงกว่าแต่กลับกินไฟมากกว่าทำให้แบตเตอรี่ใช้งานได้น้อยกว่าเท่าที่ควรจะเป็น กับพกพาได้สะดวกกว่าเนื่องจากมีน้ำหนักที่เบากว่า บางกว่าโน้ตบุ๊คที่มีขายในท้องตลาด ทำให้คนหลายคนเริ่มที่จะสนใจนำ tablet แทนที่จะพกโน้ตบุ๊คออกไปทำงานนอกบ้าน

อย่างไรก็ตามการมาของ Microsoft Office บน iPad ทำให้เราเฝ้ารออย่างมีความหวังที่ทางบริษัท Microsoft จะใจดีที่จะนำ Microsoft Office ลงบนอุปกรณ์อย่าง Android tablet เวลาผ่านไปหลายเดือน (นับไว้น่าจะราวๆ 8 เดือน) ทางบริษัทได้นำลง Android tablet จริงแต่เป็นรุ่นพรีวิวอยู่ ตอนแรกเราจะขอใช้งานรุ่นนี้แต่ติดปัญหาหนึ่งอยู่คือทางบริษัทไม่ได้เปิดให้คนไทยได้ใช้งาน ซึ่งเราคิดว่าโปรแกรมนี้ยังแก้ไขเรื่องภาษาไทยยังไม่เสร็จเรียบร้อยดีนักแต่ก็ไม่ทราบอยู่ดีว่าทำไมถึงไม่ปล่อยให้คนไทยได้โหลดไปใช้งาน

โปรแกรมนี้บน iPad ก็ไม่ได้เปิดให้คนไทยใช้งานเช่นเดียวกันแต่เราสามารถโหลดจากอเมริกาได้ แต่สำหรับเราที่ใช้งาน Android tablet กลับทำไม่ได้เนื่องจากเราใช้งาน Android tablet ที่ใช้บัญชี Google Play ประเทศไทยซึ่งไม่มีโปรแกรมนี้ให้ทดสอบ ครั้นจะเปลี่ยนไปใช้ประเทศอเมริกาก็ใช่เรื่อง เพราะเราโหลดและซื้อแอปเก็บไว้เป็นสิบทำให้เราย้ายไปประเทศอื่นก็ไม่สะดวก แล้วเราจะทำอย่างไรดีล่ะ?

สำหรับเราแล้วคิดว่าให้รอก็ไม่ดีแน่ เพราะจะไม่ทันต่อการนำโปรแกรมนี้ไปใช้งานจริง เราก็เลยเปลี่ยนไปใช้งานโปรแกรมตัวอื่นแทน เราก็เลยค้นหาโปรแกรมแนวนี้บน Google Play ก็พบโปรแกรมหนึ่งของบริษัท Google เองที่มีชื่อว่า Google Docs, Sheet, Slides เราก็ดาวน์โหลดมาใช้งาน หลังจากที่ทดลองทำพรีเซ้นต์งานโดยใช้โปรแกรมตัวนี้พบว่าใช้งานได้ดี และยังเปิดให้คนอื่นมาแก้ไข คอมเม้นต์งานได้สะดวก จุดนี้เองทำให้โปรแกรมนี้น่าใช้งานจนกระทั่งลืมโปรแกรมอย่าง Microsoft Office ไปเลย

ถึงแม้ว่าโปรแกรม Google Docs จะดีสักแค่ไหน แต่เราคิดว่าก็ยังดีไม่พอสำหรับเราอยู่ดี เนื่องจากโปรแกรมนี้ไม่รองรับการเพิ่มฟ้อนต์เอง (ต้องเพิ่มจากหน้าที่รวบรวมฟ้อนต์ของ Google เอง) กับการทำงานกับคนอื่นยังลำบากอยู่ถ้าทำงานกับคนที่ใช้โปรแกรม Microsoft Office เนื่องจากการจัดเรียงข้อความยังเพี้ยนอยู่พอสมควร (จริงๆ โปรแกรมนี้บน Mac OS X ก็ยังจัดเรียงข้อความเพึ้ยนเมื่อเทียบกับ iWork) อย่างไรก็ตามโปรแกรม Google Docs ก็ทำงานได้ดีพอสมควรเลยเชียวล่ะ!

สำหรับเราเองก็คงต้องรอความหวังที่ทางบริษัท Microsoft จะใจดีที่จะปล่อยโปรแกรมนี้ให้คนไทยใช้งานได้บ้าง อย่างไรก็ตามถึงทางบริษัทจะปล่อยโปรแกรมนี้ให้คนไทยใช้งานแล้ว เราก็รู้สึกได้เลยว่ามัน...มาช้าเกินไปนะ

หมายเหตุ บทความนี้ก็ยังเขียนโดยใช้ Android tablet เช่นเดียวกันแต่เราใช้งานผ่านเว็บเพราะโปรแกรมนี้บน Android tablet ก็ไม่เอื้อต่อการเขียนบล็อกนะ
