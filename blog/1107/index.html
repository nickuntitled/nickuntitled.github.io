เรามาลองชุบชีวิตคอมพิวเตอร์เครื่องเก่ามาใช้กันเถอะ!

December 20, 2014

Reading time ~1 minute

ท่านเคยมีคอมพิวเตอร์ใช้เมื่อไร?

หลายคนอาจจะตอบว่ามีคอมพิวเตอร์ตั้งแต่ 2-3 ปีที่แล้ว จนถึง 10 กว่าปีขึ้นไป คอมพิวเตอร์เครื่องนั้นอาจจะมีสเปคที่ธรรมดา หรือสเปคที่สูงกว่ามาตรฐานทั่วไปเมื่อเทียบกับคอมพิวเตอร์ในยุคนั้น แต่เวลาผ่านไปคอมพิวเตอร์แบบนี้อาจจะไม่แรงเท่าเดิมแล้วเนื่องจากโปรแกรมที่ใช้งานต้องการสเปคที่สูงขึ้นไปอีก

การแก้ปัญหานี้หลายคนอาจจะแก้ไขโดยการจ่ายเงินเพื่อซื้อคอมพิวเตอร์เครื่องใหม่ไปเลย แต่สำหรับเราเองคิดว่าไม่คุ้มค่ามากนัก ถ้าจะเปลี่ยนคอมพิวเตอร์เครื่องใหม่เพื่อใช้งานเพียงแค่เปิดเว็บ กับทำงานออฟฟิสทั่วๆ ไป อย่างไรก็ตามการจ่ายเงินเพื่อซื้อคอมพิวเตอร์เครื่องใหม่เพื่อเอาไปเล่นเกม หรือทำกราฟฟิกน่าจะเป็นเรื่องที่คุ้มค่ามากกว่า

ในบทความนี้เราจะไม่กล่าวถึงการจ่ายเงิน เราจะกล่าวถึงการนำคอมพิวเตอร์เครื่องเก่ามาใช้งานอินเตอร์เน็ตหรือออฟฟิสทั่วๆ ไป หรือมากกว่านั้น ถ้าเครื่องเหล่านั้นทำงานไหว การนำคอมพิวเตอร์เหล่านี้มาใช้งานต้องพิจารณาหลายปัจจัยประกอบกัน ได้แก่

อย่างแรกคือสเปค การนำคอมพิวเตอร์เครื่องเก่ามาใช้งานต้องคำนึงถึงความต้องการของระบบของโปรแกรม กับระบบปฏิบัติการที่เราต้องการจะใช้ ความต้องการของระบบจะแตกต่างกัน ตัวอย่างเช่น ระบบปฏิบัติการ Windows 8 ใช้สเปคเท่านี้แต่ Linux อาจจะใช้ต่ำกว่าหน่อยถ้าใช้งาน desktop ที่ไม่กินแรมมากนักอย่าง XFCE หรือ LXDE นอกจากนี้ถ้าต้องการใช้งาน text mode สเปคทีต้องการจะต่ำเรียดดินเลยทีเดียว

อย่างที่สองคือความต้องการ ความต้องการของแต่ละคนจะแตกต่างกันมากนัก บางคนอาจจะไม่ต้องการใช้งานคอมพิวเตอร์เครื่องเก่าแล้ว คอมพิวเตอร์เครื่องเก่าเหล่านั้นอาจจะนำไปให้ญาติ หรือนำไปขายเป็นมือสองต่อไป แต่บางคนไม่ต้องการเปลี่ยนเครื่องใหม่ เขาจำเป็นต้องใช้งานคอมพิวเตอร์เครื่องเก่าอยู่จึงนำมาอัพเกรตแล้วนำมาใช้งานได้อยู่นะ

อย่างที่สามคือเงิน บางครั้งคอมพิวเตอร์ที่ใช้อยู่มีสเปคไม่เพียงพอต่อความต้องการของระบบในโปรแกรมที่ใช้งานในชีวิตประจำวัน เราจำเป็นที่จะต้องอัพเกรต ค่าใช้จ่ายที่จะต้องอัพเกรตไม่เท่ากันขึ้นกับสิ่งที่อยากจะอัพเกรต ตัวอย่างเช่น Linux ที่ใช้ Desktop อย่าง Unity หรือ GNOME หรือ KDE ต้องการหน่วยความจำประมาณ 1 GB ขึ้นไป ถ้าเครื่องนั้นมีหน่วยความจำแค่ 512MB เราต้องจ่ายเงินเพื่อซื้อแรมมาเพิ่มอีก 512MB ราคาก็น่าจะประมาณไม่ถึงหนึ่งพันบาท

เมื่อพิจารณาแล้ว เรามาเริ่มกันเลยดีกว่าโน้ตบุ๊คที่เราจะนำมาใช้อีกครั้งเป็นรุ่น ASUS A8JN ที่น่าจะซื้อมาแล้วประมาณ 7 - 8 ปีก่อน CPU ก็เป็นรุ่น Core 2 Duo T5500 การอัพเกรตเราจะอัพเกรตแค่หน่วยความจำและ hard disk กับเปลี่ยนคีย์บอร์ด

การอัพเกรตหน่วยความจำ เราจะเพิ่มหน่วยความจำจากเดิมที่มี 1.5GB เป็น 2GB โดยนำหน่วยความจำจากคอมพิวเตอร์เครื่องเก่าที่ไม่ใช้แล้วมาใช้ กับอัพเกรต hard disk เราจะเปลี่ยน hard disk จากเดิมที่เป็นจากหมุนความจุ 100GB ที่หมุนด้วยความเร็ว 5400 รอบต่อนาทีเป็น SSD ยี่ห้อ Plextor M5P ความจุ 128GB แทน ความเร็วที่ได้พบว่าเร็วขึ้น แต่เร็วน้อยกว่าสเปคที่เขียนไว้เพราะเป็นข้อจำกัดของ SATA เองที่ยังเป็นรุ่นแรก

นอกจากการอัพเกรตแล้วยังมีการเปลี่ยนตัวคีย์บอร์ดที่เดิมกดบางปุ่มไม่ติด เรานำไปเปลี่ยนคีย์บอร์ดที่ร้านซ่อมโน้ตบุ๊คที่ฟอร์จูนชั้น 4 ตรงข้ามร้านอมร ราคาเปลี่ยนคีย์บอร์ดอยู่ที่ 1,000 บาท

เมื่ออัพเกรตและเปลี่ยนคีย์บอร์ดแล้ว เราจะลงระบบปฏิบัติการที่ต้องการ เราจะลงระบบปฏิบัติการ Windows 10 Technical Review ซึ่งให้ทดลองใช้ได้ฟรีจนถึงประมาณเมษายนที่จะถึงนี้ ก่อนอื่นเราก็จะดาวน์โหลดจากเว็บไซต์ของไมโครซอฟต์โดยตรงเสียก่อน เมื่อดาวน์โหลดเสร็จแล้ว เราก็จะเขียนลงแผ่นซีดี สั่งรีสตาร์ทใหม่แล้วเลือกให้บูตจากซีดี หรืออาจจะสั่งให้โปรแกรม Windows USB/DVD Download Tool สร้างตัวติดตั้ง USB สั่งให้เครื่องรีสตาร์ทใหม่แล้วค่อยเลือกให้บูตจาก USB ก็ได้อีกทางหนึ่ง

พอเข้าหน้าตัวติดตั้งเสร็จ เราก็ทำตามหน้าจอติดตั้งให้เสร็จเรียบร้อย เมื่อทำเสร็จแล้วเราก็จะนำคอมพิวเตอร์เครื่องเก่ามาใช้งานได้โดยที่ไม่จำเป็นต้องซื้อคอมพิวเตอร์เครื่องใหม่เลย

สำหรับผู้อ่านคิดว่าแนวทางนี้น่าสนใจไหม? ถ้าใช่ก็ลองนำบทความนี้เป็นแนวทางนะครับ
