เหตุผล 6 ข้อที่ Tumblr สามารถทำเว็บไซต์ได้มากกว่าการทำบล็อก

September 26, 2014

Tunblr คือหนึ่งใน social network ที่คนส่วนใหญ่จะเอาไว้ทำบล็อกแบบสั้นๆ ที่สามารถใส่รูป วิดีโอ คำคม และอื่นๆ ได้ หรืออาจจะทำเป็นบล็อกที่เขียนเรื่องยาวๆ ได้อีก นอกจากนี้ Tumblr ก็สามารถทำสิ่งที่ Facebook ทำไม่ได้นั่นก้คือการใส่รูปเคลื่อนไหว เป็นต้น อย่างไรก็ตามคนส่วนใหญ่มักจะเอามาทำแต่บล็อกอย่างเดียวเท่านั้น

สิ่งที่ Tumblr ทำได้ไม่ใช่เพียงแค่เอาไว้ทำบล็อกเท่านั้น วันหนึ่งเราทดลองค้นหาจาก Google ดูก็บังเอิญเจอเว็บหนึ่งที่ชื่อว่า UX Myths ซึ่งเป็นเว็บเกี่ยวกับความเชื่อที่นักออกแบบ (หรือเปล่า?) มักจะเชื่อกัน แล้วแนะนำเพื่อแก้ไขความเชื่อที่คนมักจะเชื่อกัน หลังจากที่อ่านเว็บนี้ก็เลยคิดได้ว่า Tumblr เอามาทำอะไรได้อีกนอกจากการเขียนบล็อกนั่นก็คือ

ทำให้เป็นเว็บไซต์เว็บหนึ่งได้เลย

แล้วเหตุผลที่บอกว่าทำเป็นเว็บไซต์ได้เลยคืออะไรล่ะ? คำตอบอยู่ด้านล่างแล้วครับ

เหตุผลข้อหนึ่ง Tumblr มีระบบธีม

เนื่องจากเว็บที่เราทำด้วย Tumblr จะทำออกมาเป็นเว็บให้ผู้ชมเข้ามาเยี่ยมก็เลยต้องมีหน้าตาที่แตกต่างกันขึ้นกับความต้องการของเจ้าของเว็บก็เลยมีระบบธีมขึ้นมา เพื่อให้เราปรับหน้าตาได้ตามใจชอบได้เลย โดยธีมที่มีให้นอกจากจะมีแบบฟรีแล้ว ยังมีแบบที่ต้องเสียเงินอีกด้วยซึ่งมักจะมีคุณภาพที่ดีกว่าแต่ก็ไม่เสมอไป เราสามารถเลือกธีมได้จากหน้าเว็บนี้ได้เลย แล้วถ้าถูกใจก็เลือกเอาไปใช้กับหน้าเว็บของเราได้

นอกจากนี้ถ้าเรายังไม่พอใจธีมที่มีให้ใช้ เราสามารถทำธีมได้เองเลยแต่เขียนง่ายกว่าการทำธีมของ WordPress เพราะไม่ต้องเรียนรู้โค้ด PHP ทำได้เราออกแบบหน้าเว็บได้เอง

ส่วนวิธีการสร้างธีมเองก็ดูได้จากหน้า How to create a custom HTML theme ของ Tumblr

เหตุผลข้อสอง Tumblr สามารถใส่ Google Analytics หรือ Disqus หรืออื่นๆ ตามใจชอบได้

ปกติเวลาที่เราสร้างเว็บขึ้นมาซักเว็บจะต้องใช้อุปกรณ์อย่าง Google Analytics หรือ Disqus เข้ามาช่วย แต่พอเราใช้ Tumblr แล้ว เราสามารถใช้พวกนี้ได้เช่นกัน ทำให้เราสามารถดูจำนวนคนที่เข้า คอมเม้นต์บทความที่เราสร้างขึ้นมาได้เลย อย่างไรก็ตามก็ขึ้นอยู่กับธีมของคนที่ทำอยู่ดีว่าอยากจะให้มีหรือไม่

แต่ถ้าเราอยากจะเพิ่มพวกปุ่ม Like เพจเรา หรือหน้าที่แสดง Timeline หรืออาจจะใส่ลูกเล่นอื่นๆ เราสามารถปรับแต่งโค้ดได้ด้วยตัวเองอยู่แล้วโดยเข้าหน้า Customize จากนั้นกด Edit HTML แล้วเราก็ก๊อปโค้ดมาวางลงตรงตำแหน่งที่ต้องการได้ทันที อ่ยางไรก็ตามบางธีมจะทำ minify (หมายถึงการย่อโค้ด) อาจจะทำแบบนี้ได้ยากครับ

เหตุผลข้อที่สาม Tumblr มีระบบหน้า (Pages)

เวลาที่เราใช้ WordPress หรือ Ghost จะมี Pages ไว้ใส่หน้าอย่าง About หรือหน้าต่างๆ ตามใจชอบ แต่พอเรามาใช้ Tumblr เราก็ยังสร้างหน้าขึ้นมาใหม่ได้เรื่อยๆ ตามใจชอบ หรือเราจะทำเป็นแค่ลิ้งค์ไว้เพื่อเปลี่ยนหน้าไปตามที่เราต้องการได้เลย

ฟังดูแล้วจุดนี้ทำให้เราใช้ Tumblr มากกว่าเฟสบุ๊คเพราะเราไม่ต้องยัดข้อมูลเยอะๆ ไว้หน้าเดียวกันอย่าง Timeline บนเฟสบุ๊ค

เหตุผลข้อสี่ Tumblr สามารถให้หลายคนช่วยกันเขียนได้

ปกติเราใช้ WordPress, Drupal หรือระบบอื่นๆ เราสามารถกำหนดให้คนอื่นมาช่วยกันเขียนได้ โดยให้เราลองนึกถึงเว็บอย่าง Blognone ที่ให้หลายคนช่วยกันเขียนได้นั่นเอง ซึ่ง Tumblr ก็ทำได้เช่นกันโดยเราสามารถ Invite คนให้เข้ามาช่วยเขียนได้

ส่วนวิธีการทำอ่านได้จากบทความในเว็บ Mashable

เหตุผลข้อที่ห้า ใช้โดเมนเนมของตัวเองได้

จุดนี้เป็นข้อดีอีกอย่างที่ทำให้เราสร้างเว็บไซต์โดยให้คนจำชื่อเว็บง่ายๆ นั่นเอง ปกติเวลาที่เราสมัครโฮสติ้งฟรีหรือใช้ Tumblr แบบที่เราไม่ได้ใช้โดเมนเนมของตัวเองก็จะเป็น ชื่อที่เราตั้ง.tumblr.com แต่เราใช้โดเมนเนมของตัวเองได้เลย เพียงแค่เราไปสมัครโดเมนเนมตามเว็บต่างๆ เช่น Godaddy เป็นต้น

หลังจากที่เราสมัครโดเมนเนมเรียบร้อยแล้ว ก็สามารถให้โดเมนเนมนั้นชี้เข้าไปที่เว็บที่เราสร้างใน Tumblr ได้เลยโดยจะใช้เป็นชื่อเว็บลิ้งค์ไป หรือจะใช้แค่ subdomain ของโดเมนเนมเราก็ยังได้

วิธีทำอ่านได้จากบทความ Using a custom domain name ของ Tumblr

เหตุผลข้อสุดท้าย เว็บบริษัทหลายเว็บก็ใช้ Tumblr

โดยทั่วไปเว็บบริษัทอาจจะจ้างคนข้างนอกทำระบบของเว็บไซต์เอง หรือใช้เครื่องมืออย่าง WordPress, Drupal เป็นต้นมาทำหน้าเว็บของบริษัทนั้นๆ อย่างไรก็ตามมีบางบริษัทที่ใช้ Tumblr มาทำเว็บไซต์ด้วยยย

เราสามารถดูรายชื่อบริษัทที่ใช้ Tumblr ได้จากบทความ 60 Brands Using Tumblr จากเว็บ Socialfresh ซึ่งเป็นบทความเก่าเขียนไว้เมื่อ 3 ปีที่แล้ว

จากเหตุผลทั้งหมดที่เราคิดได้ เราก็กล่าวได้เลยว่า Tumblr เหมาะกับการทำเว็บไซต์ อย่างไรก็ตามการใช้ Tumblr อาจจะไม่เหมาะที่จะทำเว็บอย่างเว็บร้านค้า เว็บรับลงทะเบียน เว็บอย่าง Facebook เป็นต้น แต่เราก็ยังพลิกแพลงให้เว็บร้านค้าทำโดย Tumblr แล้วให้ซื้อผ่านทาง Inbox ก็ยังได้เลยครับ

และการใช้ Tumblr อาจจะไม่เหมาะกับเว็บเก่า เพราะการย้ายข้อมูลจากเว็บเก่าที่ใช้พวก WordPress เข้า Tumblr ที่ทำได้ยากตัวอย่างเช่นบล็อกนี้ที่เคยคิดจะย้ายแต่กลับไม่ได้ย้ายนะแหละครับ

สรุปแล้ว การใช้ Tumblr ทำเว็บไซต์ก็เป็นแนวทางที่ดีอีกแนวทางหนึ่งครับ
