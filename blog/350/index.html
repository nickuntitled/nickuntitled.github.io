แชร์ประสบการณ์เปลี่ยนโน้ตบุ๊คจากพีซีไปแมค

Jul 29, 2014

อันนี้เขียนขึ้นหลังจากที่ใช้แมคไปแล้วประมาณ 3 เดือนนะครับ

เอาจริงๆ เดิมทีตอนแรกที่จะซื้อโน้ตบุ๊คใหม่ ผมตั้งใจจะซื้อแท็บเล็ตที่สามารถใช้เป็นคอมพิวเตอร์ได้ในตัวที่เป็นรุ่น ASUS Transformer Book T100 ตั้งแต่เดือนตุลาคมปีที่แล้ว แต่ไปๆ มาๆ ยังไงก็ไม่รู้ซื้อมาผิดรุ่นซะงั้น แถมตกรุ่นเป็นปีอีกด้วยยยยย - -"

รุ่นที่ซื้อมาผิดรุ่นเป็น ASUS Transformer Pad TF300T ที่ระยะเวลาที่ใช้เมื่อใช้แบตเตอรี่สั้นยังกับ Surface Pro รุ่นแรก (ราวๆ 4-5 ชม ถ้าไม่ได้ต่อคีย์บอร์ดพ่วงด้วย) ซึ่งตอนนี้ยังมีขายที่ IT City พันทิพย์พลาซ่า ประตูน้ำอยู่ (แต่ใครจะซื้อ?)

ทีนี้ก็พยายามใช้ซักระยะหนึ่งก็เลยอยากจะซื้อคีย์บอร์ด ตอนแรกตั้งใจว่าจะซื้อในไทยแต่กลับกลายเป็นว่าทาง ASUS ไม่ได้นำคีย์บอร์ดมาขายแยก แต่ขายรวมกับตัว tablet อยู่แล้ว ทีนี้ก็เลยหาใน Ebay เจอร้านที่ขายคีย์บอร์ดแบบ refurbished ที่ได้ในราคาที่ถูกกว่าจากปกติที่ขายประมาณ 3-4 พัน แต่ซื้อมาได้ในราคา 2 พันกว่าๆ บาทจากร้าน Buyspry

ตอนนั้นก็ซื้อมาปุ๊บรอทาง DHL ส่งมาไทยแล้วมาถึงบ้านคิดภาษีบวกค่าอื่นๆ อีก 8 ร้อยกว่าๆ บาท พอจ่ายเรียบร้อยก็เอามาใช้ก็พอใช้ได้ แต่ตัว touchpad กากกกกกกกกกมาก ก็เลยซื้อเมาส์ของ Microsoft Sculpt Comfort ที่เป็นเมาส์บลูทูธมาใช้ ก็เอามาต่อกับตัว tablet แล้วใช้ได้ดีแฮะ เอามาแทนคอมพิวเตอร์ได้ (ในระดับหนึ่ง) ใช้ไปๆ มาๆ ก็ติดขัด ทำอะไรไม่ค่อยได้มากนัก ทีนี้จะทำยังไงดีล่ะ?

ผมก็มองหาเครื่องใหม่ ตอนแรกจะซื้อ ASUS Transformer Book T100 แต่ลองจับแล้วตัวคีย์บอร์ดเล็กกว่าปกติในแนวตั้งครึ่งหนึ่งซึ่งใช้ไม่ถนัดมากกกก ก็มองข้ามรุ่นนี้ไป ลองหาหลายรุ่นแล้วไปๆ มาๆ ซื้อ MacBook Air ขนาด 11 นิ้วมาใช้ ก็เลยสั่งเพิ่มแรมไปเป็น 8GB แล้วรอทางนั้นส่งมาไทย ก็ส่งมาเรียบร้อยปุ๊บ ตั้งค่าเสร็จเรียบร้อยแล้วก็เอามาใช้ก็พบว่า

ใช้และพกพาได้สะดวกมากกกกกกกกกกกกกก

นอกจากพกพาได้สะดวกมาก ก็เป็นเรื่องแบตเตอรี่ที่อึดสุดๆ ทำงานได้ทั้งวันเลยทีนี้ 555

อย่างไรก็ตามมีข้อเสียอยู่สองอย่างคือจอเล็กไป กับไม่มีช่องเสียบ SD Card ต้องเสียบผ่าน USB เอา ตรงนี้อาจจะไม่เหมาะสำหรับบางคนที่ต้องถ่ายรูป หรือโอนไฟล์เข้า SD Card อยู่บ่อยๆ

หลังจากที่ได้แมคมาแล้ว ต่อมาก็ขาดโปรแกรมที่ใช้เป็นประจำอย่าง Microsoft Office ทีนี้จะทำยังไงดีล่ะ ตอนแรกก็เช่าบริการคลาวด์อย่าง Microsoft Azure เพราะเป็น Windows รุ่นที่ใหม่กว่าของ Amazon AWS แล้วเอามาใช้ก็พบว่า

เน็ตที่คณะกากกกกกกก

จบเลยละ 555 ทีนี้ก็เลยล้มแผนนี้ไป ต่อมาผมก็นึกขึ้นได้ว่างานส่วนใหญ่ใช้เป็นภาษาอังกฤษเป็นหลักก็เลยโหลด Microsoft Office 2011 สำหรับแมคลงเครื่องซึ่งหน้าตาก็รับไม่ค่อยได้มากนัก ตอนนี้ก็เลยใช้สลับกันไประหว่าง iWork (Pages, Keynote, Numbers) และ Microsoft Office ขึ้นกับลักษณะงานที่ต้องการ เรื่องปัญหาเอกสารเปิดมาแล้วเพี้ยนนี่ไม่ค่อยเจอมากนัก เพราะใช้เป็นภาษาอังกฤษอยู่แล้วเลยรอดตัวไป

ส่วนโปรแกรมอื่นๆ อย่าง Adobe, Brackets (ไว้เขียนโค้ด), Cyberduck, Firefox, Chrome และอื่นๆ ก็มีอยู่ทั้งสองระบบปฏิบัติการเลยไม่ค่อยมีปัญหามากนัก แต่อาจจะมีปัญหาเวลาที่อยากจะเล่นเกมบนพีซีโคตรจะฮิตซึ่งไม่มีใครทำมาลงบนแมคเลยย อย่างไรก็ตามเรื่องนี้ก็ไม่ค่อยกังวลมากนักเพราะส่วนใหญ่จะเล่นเกมบนมือถือมากกว่านะ 5555

นอกจากนี้โปรแกรมบางตัวที่ทำบนแมคที่โหลดจาก Mac App Store บางตัวก็ดีเกินบรรยายเสียจริงๆ

สิ่งที่ดีสำหรับแมคอีกอย่างหนึ่งที่ไม่อยากจะกล่าวมากนักก็คือ

แมคไม่มี Baidu PC Faster, Baidu Antivirus นะคร้าบบบบบบบ

เอาเข้าจริงโปรแกรมทั้งสองตัวนี้ก็ไม่คิดจะใช้อยู่แล้ว (ฮา) แต่ลองใช้ Search Engine อย่าง Baidu แล้วก็ใช้ได้ดีเลยล่ะ แต่ผมติด Google มากกว่า 555
