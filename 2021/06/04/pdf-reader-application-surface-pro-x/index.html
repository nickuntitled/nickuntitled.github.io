พูดถึงแอพอ่านไฟล์ PDF บน Surface Pro X

2021-06-04

PDF คือนามสกุลไฟล์เอกสารประเภทหนึ่งที่มีชื่อเต็มว่า Portable Document Format ที่พัฒนาโดยบริษัทอะโดบีที่พัฒนาโปรแกรมชื่อดังอย่าง Acrobat Reader ไฟล์ประเภทนี้เรานำไปอ่านในระบบปฏิบัติการต่าง ๆ ได้โดยที่ลักษณะตัวไฟล์สามารถแสดงผลอย่างไม่ผิดเพี้ยนไปจากเดิม รวมถึงเก็บตัวอักษร รูปภาพ รูปลายเส้นในหน้าหนังสือที่มีตั้งแต่หนึ่งหน้าไปจนถึงหลายพันหน้าเป็นต้นไป

โปรแกรมอ่านไฟล์ PDF แสดงผลไฟล์เอกสารที่สามารถเปิดเพื่ออ่าน แก้ไข จดบันทึก ไฮไลค์ลงไปในไฟล์ได้ ซึ่งโปรแกรมนี้จะพบได้ทุกระบบปฏิบัติการ ทำให้สามารถอ่านไฟล์เอกสารนี้ได้สะดวก ยิ่งบนระบบปฏิบัติการวินโดว์แล้ว ยิ่งมีโปรแกรมเหล่านี้ให้ดาวน์โหลด หรือให้ซื้อเพื่อนำไปใช้จัดการเอกสารเยอะมาก

อย่างไรก็ดี หลังจากที่เปลียนจากคอมโน้ตบุ๊คมาใช้แท็บเล็ต Surface Pro X ของไมโครซอฟท์ที่ใช้ซีพียูประเภท ARM ที่มีความแตกต่างด้านสถาปัตยกรรมกันกับซีพียู x86 อย่างอินเทล/เอเอ็มดีแล้ว ทำให้โปรแกรมที่ใช้จะต้องออกแบบมาให้ใช้กับ ARM เพื่อให้ได้ประสิทธิภาพที่ดีที่สุด แต่โปรแกรมอ่านเอกสารที่ออกแบบมาให้ใช้กับซีพียูเหล่านี้มีไม่เยอะนัก ตัวอย่างเช่น Microsoft Edge, Drawboard PDF, Nebo ที่สามารถใช้งานได้ดี

สำหรับรายชื่อโปรแกรมอื่น ๆ ที่ใช้ สามารถอ่านได้ในบทความเก่าครับ

ส่วนโปรแกรมชื่อดังอย่าง Adobe Reader DC หรือ Acrobat Pro ไม่ได้ออกแบบมาให้ใช้งานบนอุปกรณ์ชิ้นนี้ครับ เราลองอ่านในกระทู้บนเว็บของผู้ผลิตเองก็มีพูดถึงว่าจะพอร์ตลงแพลตฟอร์มนี้ แต่ยังไม่ได้บอกวันที่ชัดเจน

นอกเหนือจากโปรแกรมข้างบนนั้นแล้วยังมีโปรแกรมอื่นที่เป็น Open Source ก็มีตัวโค้ดให้ดาวน์โหลดเพื่อนำไปคอมไพล์อยู่ แต่ยังไม่ได้ลองเนื่องจากดูคู่มือการคอมไพล์โค้ด และตัวโปรแกรมที่ใช้แล้วดูยุ่งยากไปหน่อย โชคดีที่เมื่อวานเปิดหน้าเว็บเจอข่าวในเว็บไทยที่พูดถึงโปรแกรม Journal ที่ผลิตในโครงการ Microsoft Garage ที่เป็นโครงการให้พนักงานทดลองทำโครงการ/โปรแกรมที่ชอบขึ้นมา ถึงแม้ว่าจะไม่เกี่ยวกับงานหลักที่ทำอยู่ในตอนนี้

โปรแกรม Journal

เราเห็นตัวโปรแกรม Journal ที่มีให้ดาวน์โหลดฟรีจาก Microsoft Store แล้วเราเปิดเข้าไปเพื่อดาวน์โหลด ดูในแท็บ System Requirements จะพบว่าตัวโปรแกรมรองรับ ARM64 ซึ่งสามารถใช้งานบนอุปกรณ์นี้ได้ เลยดาวน์โหลดมาใช้ เรารอซักพักวินโดว์ดาวน์โหลดโปรแกรมนี้เรียบร้อย เปิดเข้าไปใช้งาน เราจะพบหน้าจอที่เรียบง่ายมาก มีรายการเอกสารให้เข้าไปเปิดอ่าน แก้ไข จดโน้ตได้สบาย นึกถึงโปรแกรมอย่าง Notability หรือ Good Notes บนไอแพดครับ

หลังจากที่เปิดเอกสารเข้าไปใช้งานแล้ว เราจะพบหน้าเอกสารให้จดโน้ตได้ รวมถึงแสดงรายการหน้าแต่ละหน้า แสดงรายการที่ทำ Bullet ไว้ รายการหัวข้อที่ขีดเส้นใต้ เป็นต้น ทำให้เปิดอ่านไฟล์ได้สะดวกขึ้นมากกว่าเดิม นอกเหนือจากนี้เราสามารถขีดตามที่กำหนดเพื่อลบข้อความ เลือกคำ ข้อความเพื่อคัดลอกและวางตามที่ต้องการ และสามารถนำเข้าไฟล์ PDF ได้

จุดนี้ทำให้เรานึกถึงโปรแกรม Nebo ที่เราใช้งานกันอยู่บนไอแพด แอนดรอยด์ รวมถึงวินโดว์ที่รองรับซีพียูที่ใช้อยู่ในอุปกรณ์นี้ด้วย

นอกเหนือจากนี้โปรแกรมสามารถซิ้งค์เอกสารเพื่อนำไปเปิดบนวินโดว์เครื่องอื่นได้เลย โดยการล็อคอินเข้าใช้บัญชีของไมโครซอฟท์เอง อย่างไรก็ดี ตัวโปรแกรมมีข้อดีแล้ว ก็ยังมีข้อเสียอยู่นะ เนื่องจากโปรแกรมดูยังใช้งานได้ไม่สมบูรณ์ท่าไร รวมถึงยังมีค้างเป็นระยะ

นอกเหนือจากโปรแกรมนี้แล้ว เราพูดถึงโปรแกรมอีกโปรแกรมที่มีชื่อว่า Nebo

โปรแกรม Nebo

Nebo เป็นโปรแกรมที่ได้รับการพัฒนาขึ้นโดยบริษัทอย่าง MyScript ที่มีความสามารถแบบเดียวกันกับในโปรแกรม Journal ที่พูดถึงก่อนหน้านี้ ตัวโปรแกรมนี้ได้รับการคอมไพล์ให้รองรับ ARM64 ซึ่งรองรับอุปกรณ์นี้เช่นกัน และสามารถดาวน์โหลดได้จาก Microsoft Store ซึ่งไม่ฟรี เสียเงินประมาณ 349 บาท

เรากดเข้าไปติดตั้งจากใน Store แล้วเปิดโปรแกรม ตัวโปรแกรมจะแสดงวิธีการใช้งาน และการแนะนำตัวโปรแกรม จากนั้นจะพบรายการเอกสารที่มีอยู่ในคอมพิวเตอร์ สามารถกดเข้าไปแก้ไข เขียน พิมพ์ได้ตามสบาย ตามสไลต์ของแอพตัวนี้ที่มีอยู่ในแอนดรอยด์ หรือไอแพดครับ

เราเปิดเอกสารเข้าไปใช้งานแล้ว เราสามารถพิมพ์งาน วาดรูป เขียนข้อความ ขีดเป็นสัญลักษณ์ตามที่โปรแกรมแนะนำเพื่อลบข้อความ ตัดขึ้นบรรทัดใหม่ รวมถึงขีดเขียนให้โปรแกรมแปลงเป็นตัวพิมพ์จากคีย์บอร์ดได้เลย ทำให้ลดการใช้งานนิ้วมือที่สัมผัสกับคีย์บอร์ดไปได้

นอกเหนือจากนี้ โปรแกรมมีความสามารถในการนำเข้าไฟล์ PDF เพื่อเปิดอ่านได้อีก ในภาพเราอ่านบทความทางวิชาการชิ้นหนึ่งอยู่ เราสามารถขีดเขียนเพื่อเน้นข้อความ จดบันทึกได้เลย และที่ดีอีกจุดคือโปรแกรมสามารถสั่งให้ซิ้งค์เอกสารบนคลาวด์ ทำให้เราเปิดอ่านบนแพลตฟอร์มอื่นได้ โดยใช้บัญชีอื่นนอกเหนือจากไมโครซอฟท์ได้ครับ

ตัวโปรแกรม Nebo ถือว่าใช้งานได้ดี ไม่มีปัญหาอะไร เพียงแต่จุดที่สังเกตของทั้งสองโปรแกรมคือ ตัวโปรแกรมยังรองรับภาษาไทยได้ไม่ดีเท่าไร ตอนที่เราเขียนแล้วกดเพื่อแปลงให้เป็นตัวพิมพ์ ทั้งสองโปรแกรมยังทำแบบนี้ไม่ได้ แต่เราสามารถขีดเขียนข้อความเป็นภาษาไทยได้ปกติ ก็คงไม่มีปัญหา หวังว่าทางผู้ผลิตจะทำให้รองรับการใช้งานภาษาไทยครับ

สรุป

ไฟล์ PDF เป็นไฟล์เอกสารที่สามารถอ่านได้ทุกแพลตฟอร์ม โดยเฉพาะวินโดว์ที่มีโปรแกรมอ่านเอกสารแบบนี้เยอะมาก เพียงแต่บน Surface Pro X มีโปรแกรมแบบนี้น้อยกว่า อย่างไรก็ดี เราสามารถใช้งานโปรแกรม Microsoft Edge, Drawboard PDF, Journal, Nebo เป็นต้น เพื่ออ่านไฟล์ลักษณะนี้ได้

สำหรับผู้อ่านที่สนใจ สามารถดาวน์โหลดเพื่อนำมาใช้ติดตั้งบนคอมพิวเตอร์ได้นะครับ เราแนะนำโปรแกรม Nebo มากกว่าโปรแกรมแรก เนื่องจากใช้งานได้ดี เสถียร รองรับบัญชีอื่นนอกเหนือจากไมโครซอฟท์ และมีความสามารถที่เพียงพอต่อการใช้งานครับ

ส่วนใครจะเอาไปเรียนออนไลน์ที่เปิดเทอมอยู่ในขณะนี้ ก็เอาไปใช้ได้เลยครับ

เพิ่มเติม

มีแอพอื่นที่น่าสนใจนอกเหนือจากที่กล่าวไว้ข้างบนก็คือแอพ Xodo PDF Reader & Editor for Windows ที่มีให้ดาวน์โหลดใน Microsoft Store ได้ฟรี และรองรับการทำงานบน Surface Pro X เช่นกันครับ แต่ยังไม่ได้ลองเพราะซื้อแอพ Nebo มาใช้ก่อนแล้วครับผม
