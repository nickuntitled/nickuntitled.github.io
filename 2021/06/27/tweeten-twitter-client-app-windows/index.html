Tweeten หนึ่งในแอพทวิตบน Windows ที่ใช้อยู่

2021-06-27

บนระบบปฏิบัติการอะไรก็ตาม ไม่ว่าจะเป็นแอนดรอยด์ วินโดว์ แมคโอเอส ไอโอเอส หรืออื่น ๆ สิ่งที่จำเป็นต่อการตัดสินใจซื้อเพื่อนำไปใช้งานคือแอพ (App) ครับ

แอพที่เราใช้กันบ่อย ๆ จะเป็นแอพโซเชียลอย่างเฟสบุ๊ค ทวิตเตอร์ อินสตาแกรม ไลน์ ซึ่งมีให้ดาวน์โหลดมาใช้งานได้อย่างดีเกือบทุกระบบปฏิบัติการครับ ยกเว้นระบบปฏิบัติการวินโดว์นี่แหละครับ ที่ดาวน์โหลดมาใช้งานได้ก็จริง แต่เป็นเพียงแอพที่เปิดหน้าเว็บไซต์เพื่อใช้งานเท่านั้น ใช้งานได้ระดับหนึ่ง แต่มันไม่เหมือนใช้งานบนแอนดรอยด์หรือแมค หรือไอโอเอสครับ

ทีนี้เราเลยค้นหาแอพอื่นที่ทำงานทดแทนแอพที่ผลิตมาจากผู้จัดทำโซเชียลเอง เราค้นหาในไมโครซอฟท์สโตร์ก็พบว่ามีแอพหนึ่งที่ดูแล้วน่าใช้งาน ที่มีชื่อว่า Tweeten

Tweeten

Tweeten เป็นโปรแกรมใช้งานทวิตเตอร์บนระบบปฏิบัติการวินโดว์ แมค กูเกิลโครมที่รองรับซีพียูสถาปัตยกรรม x86 และ ARM/ARM64 แอพนี้ส่วนตัวคิดว่าได้แรงบันดาลใจจาก Tweetdeck ที่แสดงผลเป็นคอลัมน์ที่มีหัวข้อแตกต่างกันออกไป แอพนี้สามารถดาวน์โหลดได้จากหน้าไมโครซอฟท์สโตร์ แต่จะดาวน์โหลดจากหน้าเว็บผู้ผลิตก็ทำได้เช่นกัน

ของเราติดตั้งผ่านไมโครซอฟท์สโตร์จะได้ตามภาพด้านล่างครับ

ติดตั้งโปรแกรมเสร็จปุ๊บแล้วเปิดแอพ Tweeten

เข้าสู่ระบบโดยใช้บัญชีทวิตเตอร์เดิม แล้วเริ่มต้นการทำงาน ในหน้าจอของแอพที่แสดงผลหลายคอลัมน์ตามภาพด้านล่างนี้ครับ

ในหน้าจอนี้เราสามารถกำหนดได้ว่าแต่ละคอลัมน์แสดงผลอะไรบ้าง จากการกดปุ่ม ADD COLUMN ที่มุมล่างซ้ายของจอ จะได้หน้าจอที่ให้เลือกชนิดคอลัมน์ตามภาพด้านล่างนี้

กดเลือกชนิดคอลัมน์ที่ต้องการ โดยเราจะอธิบายบางคอลัมน์ ตัวอย่างเช่น

  • Home คือคอลัมน์แสดงหน้า News Feed
  • User แสดงผลทวิตของเราเอง
  • Notification แสดงผลการแจ้งเตือน
  • Search แสดงหน้าค้นหา
  • List แสดง List ที่เราสร้างขึ้นในทวิตเตอร์
  • Trending แสดงแฮชแทคที่กล่าวถึงเป็นจำนวนมากอยู่ในขณะนั้น

ของเรากำหนดหน้าจอให้แสดงผล Home, Activity, Trending, Notifications, User ของเรา และ User ที่เราติดตาม และสนใจ ที่เราสามารถเพิ่มคอลัมน์ได้จากการกดปุ่ม ADD COLUMN ที่มุมล่างซ้ายของหน้าจอ กดเลือกชนิดคอลัมน์ที่ต้องการ แล้วแอพจะเพิ่มคอลัมน์ให้เองตามภาพในย่อหน้าก่อนหน้านี้ครับ

รีวิวหน่อยนึง

พอได้ทดลองใช้ระยะหนึ่งพบว่าตัวแอพเรียนรู้เพื่อที่จะใช้งานได้ไม่ยาก เร็ว ทำงานได้ดี ไม่ค้างอะไร แถมสามารถทำงานบนคอม Surface Pro X ที่ใช้ซีพียูสถาปัตยกรรม ARM64 ได้อีก ทำให้ใช้งานโปรแกรมแบบเนทีฟที่ใช้ความสามารถอย่างเต็มที่ ไร้ความกังวลจากการใช้งานแอพบนสถาปัตยกรรม x86 ที่ต้องผ่านการแปลงโค้ดอะไรบางอย่างก่อน ทำให้ประสิทธิภาพที่ได้ต่ำลงครับ

ราคา

ส่วนราคาของแอพนี้ ฟรี ไม่มีค่าใช้จ่ายแต่อย่างไร อย่างไรก็ดี ถ้าเราใช้งานแอพนี้แล้วพบว่าถูกใจ อยากจะสนับสนุนผู้พัฒนาแอพ เราสามารถกด Donate เพื่อบริจาคเงินได้ครับ

ส่วนผู้อ่านที่สนใจจะใช้งานแอพนี้บนวินโดว์ แมค เว็บ หรืออื่น ๆ เราแนะนำแอพนี้นะ
