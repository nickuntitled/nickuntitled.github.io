ลงงานวิ่ง Virtual Run ครั้งแรก

20.04.2021
Share: 

เดือนนี้มีงานวิ่ง Virtual Run ที่ให้วิ่งระยะทาง 150 กิโลเมตร แล้วให้แต่ลงคนวิ่งแล้วส่งระยะที่วิ่งได้จากการบันทึกผ่านโปรแกรม Strava โดยโปรแกรมนี้นึกถึงเครือข่ายสังคมออนไลน์ของนักกีฬาที่ให้คนโพส คอมเม้นต์ แสดงสถิติได้ แล้วเดิมทีเราลงงานวิ่งบ้าง แต่วิ่งระยะทาง Fun Run กับ Mini Marathon ที่วิ่งในระยะทาง 5 กับ 10 กิโลเมตร แต่ก็ไม่ได้จริงจังเท่าไรนัก

ต่อมาก็ห่างหายไประยะหนึ่ง แล้วมาวิ่งบนลู่วิ่งได้ประมาณ 1-2 เดือน จากนั้นเห็นโพสในเฟสบุ๊คของมิตรสหายท่านหนึ่งที่โพสให้ลงสมัครวิ่งแบบ Virtual Run (วิ่งแบบเก็บระยะ ถ้าครบกำหนดก็จะได้รางวัล) แต่งานนี้ต่างกับงานอื่นตรงที่ใครวิ่งจบบ๊วยจะเป็นฝ่ายเลี้ยงอาหารแนวชาบูสุกี้อะไรประมาณนี้ เราเห็นแล้วเป็นระยะสะสมที่ไม่ยากจนเกินไปนักเนื่องจากสามารถวิ่งต่อเนื่องในระยะเวลาที่กำหนดได้ และการตั้งเป้าหมายการวิ่งต้องมีระยะเวลาที่จำกัด เลยลงทะเบียนไป

หลังจากที่ลงทะเบียนไปแล้ว พอดีวันที่กำหนดก็เริ่มต้นวิ่งเก็บระยะได้เลย โดยเราค่อย ๆ วิ่งไปเรื่อย ๆ วิ่งเพื่อสุขภาพประมาณ 5-10 กิโลเมตร ก็เก็บไปได้อยู่นะ ไม่ได้เหนื่อยอะไร พูดได้สบาย โดยสถานที่วิ่งก็วิ่งหลายที่ ตั้งแต่ในมหาวิทยาลัย แถวบ้าน วิ่งในซอย วิ่งในเมือง ก็คือวิ่งได้หมดทุกที่ตราบใดที่ไม่ได้วิ่งบนลู่วิ่งแค่นั้น ส่วนคนอื่นก็วิ่งระยะฮาร์ฟ ฟูลมาราธอน หรืออัลตร้าก็วิ่งไป โดยในแต่ละวันก็จะรวบรวมสถิติการวิ่งว่าเราวิ่งได้เท่าไรแล้ว ซึ่งเป็นแรงกระตุ้นได้ดีว่าเราควรจะวิ่งให้มากกว่านี้

ทีนี้ดูจากสถิติเราก็ไปได้เรื่อย ๆ ใช้เวลาไม่นานนัก (ประมาณ 2 สัปดาห์นิด ๆ) ก็วิ่งสะสมจนได้ระยะ 150 กิโลเมตรได้ เพียงแต่ถ้าวิ่งเร็ว หรือวิ่งเยอะไปหน่อยก็จะมีอาการเมื่อยบริเวณฝ่าเท้าและขาคิดว่ารองเท้าน่าจะเป็นส่วนหนึ่งที่ทำให้เกิดอาการ และกล้ามเนื้อขา และสะโพกตึงไปหน่อย ต้องยืดกล้ามเนื้อร่วมกับพักเป็นระยะ โดยทำตามภาพด้านล่างได้เลย

สุดท้าย สิ่งที่ได้จากงานวิ่งงานนี้ก็เป็นการออกกำลังกาย ร่วมกับการลดน้ำหนักซึ่งเป็นผลพลอยได้ โดย

  • 70% มาจากการปรับการกินอาหาร
  • 30% มาจากการวิ่งนี่แหละ

ในงานนี้น้ำหนักลงไม่เท่าไร แต่ไขมันลงจาก 24% เป็น 22% และกล้ามเนื้อเพิ่มขึ้นซึ่งเป็นการลดน้ำหนักที่ถูกต้องแล้ว สำหรับใครที่อยากจะลดน้ำหนักตัว การออกกำลังกายและการปรับการกินอาหารให้ดีขึ้นก็เป็นทางเลือกที่โอเคสำหรับการลดน้ำหนักครับผม และใครอยากวิ่งช่วงนี้ ก็ลงงาน Virtual run ไว้ดีกว่า ลดความเสี่ยงต่อโควิดด้วยนะ
