วิธีใช้ WARP+ Unlimited 1 บัญชี หลายเครื่อง

Published: 17 September 2021
Share:

บริการ 1.1.1.1 with WARP เป็นบริการที่ออกมาโดยบริษัท Cloudflare ที่ให้บริการ DNS, CDN และอื่น ๆ ที่หลายเว็บไซต์ใช้บริการอยู่ตอนนี้ครับ บริการนี้มีที่มาเนื่องมาจากต้องการให้ใช้งานอินเตอร์เน็ตได้เร็วขึ้นตามที่กล่าวไว้ในหน้าเว็บว่า

WARP is built on the same network that has made 1.1.1.1 the fastest DNS resolver on Earth.

ข้อความในหน้าเว็บ 1.1.1.1 with WARP ของ Cloudflare

และบริการนี้ยังอนุญาตให้เราใช้งานได้ปลอดภัยขึ้น รวมถึงให้เราเพิ่มความเป็นส่วนตัวด้วย เนื่องมาจาก DNS สามารถบันทึก Log ข้อมูลการใช้งานได้ว่าเราใช้งานที่ไหนบ้าง บางที่อาจจะเอาข้อมูลไปขาย รวมถึงเอาข้อมูลในใช้ทำโฆษณาอะไรบ้าง นอกเหนือจากนี้ทาง ISP หรือผู้ให้บริการอินเตอร์เน็ตยังเห็นข้อมูลของเราได้ ถ้าเว็บนั้นไม่ได้เข้ารหัสโดย HTTPS

หลักการนี้ ทำให้เรานึกถึงบริการ VPN (Virtual Private Network) ครับ ที่เรามักจะกล่าวถึงเวลาที่เข้าเว็บไซต์ที่โดนบล็อก ซึ่งทางบริการ WARP จะทำให้เราโดยเราไม่จำเป็นต้องมีความรู้ว่าสิ่งนี้คืออะไร [1] ตามที่กล่าวไว้ว่า

WARP was built on the philosophy that even people who don’t know what “VPN” stands for should be able to still easily get the protection a VPN offers.

ข้อความในหน้าบทความของ Cloudflare

บริการ WARP+ และการสมัคร WARP+ Unlimited

ต่อมาเราจะมาพูดถึงบริการ WARP+ บริการนี้จะเป็นบริการ VPN คล้ายกันกับ WARP ที่เรากล่าวถึงก่อนหน้า อย่างไรก็ดี บริการนี้มีข้อที่แตกต่าง บริการนี้จะใช้งานเครื่องมือ Argo Smart Routing ที่ช่วยทำให้เราใช้งานอินเตอร์เน็ตได้โดยมีค่า Latency ที่ต่ำที่สุดเท่าที่ทำได้ เพื่อให้เราสามารถเข้าถึงเว็บไซต์ได้โดยใช้ระยะทางที่สั้นที่สุด แต่บริการนี้มีข้อจำกัดที่ปริมาณข้อมูล ดังนั้นเราใช้งานแบบ Unlimited จะดีกว่า [2]

การสมัครทำได้ไม่ยาก เวลาที่เราเปิดหน้าแอพ 1.1.1.1 บน iOS หรือ Android เพื่อใช้งาน เราสามารถกดสมัครสมาชิกบริการ WARP+ Unlimited ได้เลย โดยค่าใช้จ่ายรายเดือน 59 บาทครับ (ซึ่งเทียบกับบริการที่เราใช้อย่าง ProtonVPN ที่เราเคยใช้ที่มีราคาประมาณ 300 กว่าบาท ถูกไปเลย)

การใช้งานบัญชีที่สมัครบริการนี้บนเครื่องอื่น

เมื่อสมัครเสร็จแล้ว เราต้องการเปิดใช้งานสิ่งนี้ที่คอมพิวเตอร์ หรืออุปกรณ์มือถือเครื่องอื่นจะทำอย่างไร?

สิ่งนี้ทำได้ไม่ยากครับ แต่ก่อนอื่น เราต้องมาคัดลอกรหัส Account ที่เราใช้งานอยู่ก่อน ขั้นตอนแรก เราเปิดเข้ามาที่แอพ 1.1.1.1 เลือกที่มุมบนขวา

เมื่อเลือกเสร็จแล้ว เราจะเข้ามาที่หน้าการตั้งค่า ให้เลือกไปที่ Account

แล้วเลือกไปที่ Key เราจะเจอรหัสหลายหลักตรงช่อง Key กดเข้าไป จะพบหน้าจอ License Key ให้คัดลอกรหัสบัญชีตรงช่อง Key ไว้ครับ

การเปลี่ยนบัญชีบนอุปกรณ์มือถือ

กรณีที่เราเปิดใช้งานอุปกรณ์ iOS หรือ Android เครื่องอื่น ให้เลือกที่ Change key ที่ช่องด้านล่างในหน้าจอ License Key นี้ แล้วใส่รหัสที่เราบันทึกไว้

แค่นี้ เราสามารถใช้งานบริการ WARP+ Unlimited ได้เลยครับ

การเปลี่ยนบัญชีบนคอมพิวเตอร์

แต่ถ้าคนที่ใช้งานบนคอมพิวเตอร์ เราสามารถใช้บัญชีมากมือถือที่เราสมัครได้เช่นกัน

ส่วนผู้ใช้งานแมค ผู้ใช้สามารถใช้งานบริการนี้ได้เลยครับ เนื่องจากมีให้ดาวน์โหลดโปรแกรมนี้บนเว็บไซต์ของ Cloudflare เอง

ส่วนคนที่ใช้ Windows 10 on ARM บริการนี้ยังใช้ไม่ได้ เนื่องมาจากโปรแกรมไม่รองรับครับ

ทำได้โดยเลือกที่สัญลักษณ์ไอคอนของ 1.1.1.1 ก่อน เราจะพบหน้าตาแบบนี้

กดที่ฟันเฟือง จะปรากฏหน้าต่างแบบนี้ขึ้นมา เลือกไปที่ Account

เมื่อเลือกแล้ว จะปรากฏหน้าจอนี้ ให้เรากดที่ Use Different Key แล้วใส่รหัสที่เราบันทึกไว้จากอุปกรณ์มือถือที่เราสมัครสมาชิกครับ

สำหรับคนที่เปลี่ยนบัญชีแล้ว เราสามารถใช้งานบริการ WARP+ Unlimited ได้แล้วครับ อย่างไรก็ดี บัญชีหนึ่งบัญชี สามารถใช้งานได้เพียง 5 เครื่องเท่านั้นครับ [3]

สำหรับคนที่สนใจ สามารถสมัครสมาชิกได้ครับ เราใช้งานบริการนี้อยู่ ก็สามารถเปิดหน้าเว็บที่เราต้องการได้อย่างปลอดภัยขึ้น รวมถึงเข้าหน้าเว็บบางหน้าที่ถูกบล็อกได้เช่นกัน และนอกเหนือจากนี้สามารถใช้งานได้เร็วไหม อันนี้คงต้องใช้งานไปสักระยะ ก็จะเห็นผลครับ
