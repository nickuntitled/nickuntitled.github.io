Surface Pro X กับแอพที่ใช้งานอยู่ในขณะนี้

20.05.2021
Share: 

Surface Pro X เป็นแท็บเล็ตที่ผลิตโดยบริษัทไมโครซอฟท์ที่ใช้ซีพียู ARM ที่ผลิตโดย Qualcomm อย่าง Microsoft SQ1 (จริง ๆ มันมี SQ2 แต่ประสิทธิภาพไม่ต่างอย่างมีนัยสำคัญแบบ M1 เลยไม่กล่าวถึง) แท็บเล็ตนี้ใช้ระบบปฏิบัติการ Windows 10 แบบที่ใช้บนคอมพิวเตอร์ส่วนบุคคลตามปกตินี่แหละ เพียงแต่ใช้ซีพียูคนละแบบแค่นั้น

ตอนเปิดหน้าเว็บ มันจะมีกรุ๊ปในเฟสบุ๊คอย่าง Surface Thailand ที่พบหลาย ๆ คนเข้ามาโพสเกี่ยวกับ Surface ตามปกติ แต่อย่างไรก็ดีจะพบว่ามีขายรุ่นนี้ทิ้งเยอะมากเพราะไม่ชินกับการใช้งาน หรือไม่ถูกใจ หรือมีปัญหาซอฟต์แวร์อะไรก็ตาม แต่ของเราศึกษามาก่อนแล้วว่ามีโปรแกรมที่เราใช้งานบนนี้ได้ และมีฟังก์ชันอย่าง Windows Subsystem for Linux (WSL) อยู่แล้ว

เอาวะ เอาติดต่อคนขายคนนึงในกรุ๊ปไป แล้วนัดไปดูเครื่องก็ไม่มีปัญหาอะไร เพียงแต่ประกันใกล้ ๆ หมดและ ก็ซื้อไป ราคาที่ซื้อถูกกว่าที่ขายมือหนึ่งไป 15,000 บาท และถูกกว่าค่าทำกายภาพคอ บ่า ไหล่ หลังจากการแบกโน้ตบุ๊ค ASUS Vivobook + iPad Pro ก็โอเค Shut up and take my money

พอมาถึงที่บ้านก็ติดตั้งอะไรเสร็จเรียบร้อย เปิดใช้งานก็ใช้งานเหมือน Windows 10 ปกตินี่แหละฮะ แล้วปัญหาที่พบเจอในกรุ๊ปนั้นก็คือมันไม่มีโปรแกรม หรือไม่ชินกับเครื่อง แต่ของเราเอามาใช้เปิดเว็บ ทำเอกสาร เขียนโปรแกรม รีโมทไปเซิร์ฟเวอร์ หรืออาจจะตัดต่อภาพ และอื่น ๆ ก็ทำได้ไม่ยากอะไร ตามด้านล่างนี้ครับ

งานเอกสาร

งานเอกสารเป็นสิ่งที่สำคัญมากเป็นอันดับต้น ๆ ของผู้ใช้ซึ่งโดยปกติเราจะใช้งานโปรแกรมอย่าง Microsoft Office ใช่ไหมครับ ทีนี้ดีอยู่ที่ว่า Windows 10 รุ่นนี้รองรับการเปิดแอพ x86 ที่เป็น 32-bit (64-bit จะมีมาทีหลัง) เลยลงใช้งานได้สบาย ไม่มีค้างอะไรเลย ทำงานได้ดีอีกด้วย

ต่อมาปัญหาที่สำคัญคือการปริ้นท์เอกสาร ของเราไม่ได้มีปัญหาอะไรนะ เอา Surface Pro X ต่อปริ้นท์เตอร์ปริ้นท์ผ่าน Wi-Fi ได้ตามปกติ ไม่มีปัญหาอะไร แถมเวลาปริ้นท์เอกสารก็สวยงามตามเนื้อผ้าปกติเลย

เปิดเว็บ

การเปิดเว็บเราใช้ผ่านเว็บเบราวเซอร์ใช่ไหมครับ ทีนี้ดีอยู่ที่ว่า Microsoft Edge ได้รับการพัฒนาให้ทำงานดีกว่าเดิมล้านปีแสง ทีนี้เลยลงใช้งานได้ปกติ ไม่มีปัญหาอะไร ใช้งานได้ลื่นเหมือนที่เปิดบนคอมพิวเตอร์ทั่วไปตามปกติเลยครับ ส่วนโปรแกรมอื่นอย่าง Google Chrome, Mozilla Firefox เราไม่ได้ใช้เลย

ส่วนโปรแกรมอย่าง Twitter, Instagram, LINE และ Zoom ก็สามารถลงโปรแกรมใช้งานได้ปกติเลยผ่าน Microsoft Store ยกเว้น Zoom ที่เราโหลดมาติดตั้งเอง ก็ไม่ได้มีปัญหาอะไร แต่ถ้าปล่อยมาให้รองรับ 64-bit อย่างเดียวก็รอทาง Microsoft ปล่อยอัพเดทมาทีหลังอยู่ดี

นอกเหนือจากนี้โปรแกรมอย่าง Spotify, Netflix เราสามารถดาวน์โหลดผ่าน Microsoft Store เพื่อใช้ฟังเพลง และดูหนัง ดูซีรีส์บนนี้ได้เช่นกันครับ

เขียนโปรแกรม

อันนี้ขึ้นกับว่าเราจะเขียนโปรแกรมอะไร ของเราเขียน Front-end, Back-end หรือเขียนทำงานวิจัยด้าน AI อยู่ ก็ใช้ Node.js, Python ผ่าน Windows Subsystem for Linux (WSL) ที่ลงระบบปฏิบัติการ Ubuntu ได้ตามปกติ แล้วใช้ IDE ที่มีชื่อว่า Visual Studio Code ที่รองรับการทำงานบน ARM เรียบร้อยแล้ว ไม่มีปัญหาอะไร

ส่วน .NET เท่าที่ค้นในเว็บพบว่าทาง Microsoft เองปล่อยเครื่องมือมาให้ใช้งานบนนี้ได้แล้ว แต่ยังไม่ได้ลอง อย่างไรก็ดีอาจจะติดปัญหาโปรแกรมเฉพาะทางบางตัวเช่น Docker หรืออื่น ๆ คงอาจจะต้องรอทางผู้ผลิตพอร์ตมาอีกที หรือถ้าไม่มี อาจจะลองพอร์ตมาใช้เอง

นอกเหนือจากนี้การรีโมทไปยังเซิร์ฟเวอร์ เราทำได้ไม่มีปัญหาอะไรเพราะโปรแกรมอย่าง Putty ก็ออกมารองรับ ARM แล้ว (แต่ยังไม่ใช่รุ่นสมบูรณ์) ก็ต่อเข้าเซิร์ฟเวอร์แล้วทำงานได้เช่นกัน

สรุป

โดยสรุป จากทั้งหมดที่กล่าวช้างบนนี้เราใช้งานบน Surface Pro X สักระยะหนึ่งก็ไม่มีปัญหาอะไร มีโปรแกรมที่รองรับการใช้งานของเราได้หมด ส่วนโปรแกรมที่ Open Source แต่ไม่ได้คอมไพล์ลงบน ARM นี่เดี๋ยวลองพอร์ตมาใช้เองดีกว่า เพราะมีคนสอนการพอร์ตบนอินเตอร์เน็ตเต็มไปหมด หรือถ้ามันไม่มีโปรแกรมบนนี้ จริง ๆ แล้วนี่ เราก็เขียนแอพใช้เองก็เป็นทางเลือกที่ดีอีกทางหนึ่งเลยนะ เพราะไลบรารีอย่าง Electron ก็ออกมารองรับ ARM แล้ว เราสามารถเขียนโดยใช้ภาษาที่ถนัดอย่าง JS และอื่น ๆ ได้เลย อิอิ
