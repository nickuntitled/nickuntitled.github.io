August 02, 2014Nick Untitled Reading time ~1 minute

ย้ายไฟล์ และโฟลเดอร์จำนวนมากไป server เครื่องหนึ่งไปอีกเครื่องหนึ่ง

วันนี้ได้ทำภาระกิจที่ (โคตรจะ) ยิ่งใหญ่เลยก็คือย้ายข้อมูลจากเครื่อง server เครื่องเก่าไปยังเครื่องใหม่ ฟังดูแล้วน่าจะเป็นงานที่ไม่ยากซักเท่าไรนัก ก็แค่ย้ายไม่กี่ไฟล์ก็น่าจะเพียงพอแล้ว

ใช่ครับ งานอาจจะฟังดูไม่ยากนักกรณีที่ถ่ายโอนไฟล์แค่ไม่กี่ไฟล์ที่มีขนาดไม่ใหญ่มาก แต่ในกรณีนี้เราย้ายข้อมูลทั้งหมดประมาณ 2 ล้านกว่าๆ ไฟล์ แต่ละไฟล์ก็มีขนาดที่แตกต่างกันไปขึ้นกับคนที่อัพโหลด รวมๆ แล้วไฟล์ทุกไฟล์ใช้พื้นที่ไปประมาณ 200 - 300 GB ความรู้สึกตอนนั้นก็ประมาณว่า

ม่ายยยยยยยยยยยยยยยยยย

แล้วจะยังไงดีล่ะ เราลองทำหลายวิธีมาก่อนแล้ว ได้แก่

ทำผ่าน FTP

ทำได้โดยให้ก๊อปไฟล์ผ่านทาง FTP นี่แหละ ตอนนั้นเราใช้ mc (Midnight Commander) เข้าไปในโฟลเดอร์ที่ต้องการอัพโหลดโดยใช้

cd ftp://username@domainname.com - เปลี่ยน username และ domainame.com เป็นอย่างอื่นก็ได้ตามต้องการ

ทีนี้เมื่อเข้ามาแล้วเราก็จะก๊อปปี้ไฟล์ลงไปใน server อีกเครื่องหนึ่งทำได้โดยเลือดไฟล์หรือโฟลเดอร์ที่ต้องการแล้วกดปุ่ม F5 หลังจากนี้เราก็จะปล่อยให้โปรแกรมก๊อปไฟล์นะครับ แต่ในกรณีของเราที่กีอปไฟล์จำนวนมากทีนี้เวลาก๊อปไฟล์เป็นจำนวนมากอาจจะไม่สะดวก หรืออาจจะทำให้เห็นว่าโปรแกรมแม่งค้างไปแล้ววววนั่นแหละ จริงๆ ก็สามารถเช็คได้ว่าโปรแกรมทำอะไรอยู่ โดยเราจะใช้

strace -p PID

PID คือ Process ID ของโปรแกรมนั้นทำได้โดยเช็คใน ps [ชื่อโปรแกรม])

ทำผ่าน rsync

ทีนี้เราก็จะใช้อีกวิธีหนึ่งก็คือการก๊อปไฟล์จำนวนมากผ่านโปรแกรมที่ชื่อ rsync วิธีนี้ก็ทำได้โดย

rsync -av [path ของโฟลเดอร์ต้นทาง] [path ของโฟลเดอร์ปลายทาง]

ส่วน -av จะอธิบายในด้านล่างนี้นะครับ

  • a (achive) ให้ก๊อปไฟล์และ directories ทั้งหมด
  • v (verbose) ให้แสดงรายละเอียดของไฟล์และ directories

ข้อดีของโปรแกรมนี้นอกจากจะก๊อปไฟล์ได้ทีละเป็นจำนวนมากแล้ว ยังทำผ่าน ssh กับ backup ข้อมูลได้ด้วยนั่นเอง ซึ่งเราทำผ่าน ssh โดยพิมพ์ดังนี้

rsync -av [path ของโฟลเดอร์ต้นทาง] username@host:[path ของไฟล์ปลายทาง]

ทีนี้เราก็จะก๊อปปี้ไฟล์จำนวนมากผ่านทาง ssh ได้เรียบร้อยแล้ว แต่ในกรณีของเราก๊อปไฟล์เป็นจำนวนมาก และความเร็วของสายแลนที่ช้า (100 MBps) เลยต้องใช้เวลานานหลายชั่วโมงกว่าจะก๊อปปี้ให้เสร็จเรียบร้อย

วิธีเช็คว่าก๊อปเสร็จหรือยัง

ระหว่างที่ทำการก๊อปปี้ไฟล์อยู่ก็เช็คได้ว่าก๊อปเสร็จหรือยัง ของเราก็จะเช็คว่าไฟล์ระหว่างเครื่องต้นทางกับปลายทางมีจำนวนที่เท่ากันหรือเปล่าโดยใช้

 ls -1 [path ของโฟลเดอร์ที่เราต้องการ] | wc -l

เมื่อพิมพ์เสร็จแล้วกด Enter จะขึ้นจำนวนไฟล์ที่มีในโฟลเดอร์ตาม path นั้นเอง

สำหรับ Windows

จริงๆ โปรแกรม FTP กับ rsync ก็มีบน Windows เช่นกัน แต่ตัว FTP อาจจะไม่ต้องทำผ่าน Command Line แต่ทำผ่านโปรแกรมบน Windows เลยเช่น FileZilla, Cyberduck และอื่นๆ ครับ

หมายเหตุ

วิธีที่พูดถึงอันนี้พูดจากที่เคยทำนะครับ อาจจะไม่ถูกต้องซะทีเดียว ลองเช็คข้อมูลเพิ่มเติมจากเว็บอื่นอีกทีนะ

แชร์ประสบการณ์เปลี่ยนโน้ตบุ๊คจากพีซีไปแมค

อันนี้เขียนขึ้นหลังจากที่ใช้แมคไปแล้วประมาณ 3 เดือนนะครับ

เอาจริงๆ เดิมทีตอนแรกที่จะซื้อโน้ตบุ๊คใหม่ ผมตั้งใจจะซื้อแท็บเล็ตที่สามารถใช้เป็นคอมพิวเตอร์ได้ในตัวที่เป็นรุ่น ASUS Transformer Book T100 ตั้งแต่เดือนตุลาคมปีที่แล้ว แต่ไปๆ มาๆ ยังไงก็ไม่รู้ซื้อมาผิดรุ่นซะงั้น แถมตกรุ่นเป็นปีอีกด้วยยยยย - -"

รุ่นที่ซื้อมาผิดรุ่นเป็น ASUS Transformer Pad TF300T ที่ระยะเวลาที่ใช้เมื่อใช้แบตเตอรี่สั้นยังกับ Surface Pro รุ่นแรก (ราวๆ 4-5 ชม ถ้าไม่ได้ต่อคีย์บอร์ดพ่วงด้วย) ซึ่งตอนนี้ยังมีขายที่ IT City พันทิพย์พลาซ่า ประตูน้ำอยู่ (แต่ใครจะซื้อ?)

ทีนี้ก็พยายามใช้ซักระยะหนึ่งก็เลยอยากจะซื้อคีย์บอร์ด ตอนแรกตั้งใจว่าจะซื้อในไทยแต่กลับกลายเป็นว่าทาง ASUS ไม่ได้นำคีย์บอร์ดมาขายแยก แต่ขายรวมกับตัว tablet อยู่แล้ว ทีนี้ก็เลยหาใน Ebay เจอร้านที่ขายคีย์บอร์ดแบบ refurbished ที่ได้ในราคาที่ถูกกว่าจากปกติที่ขายประมาณ 3-4 พัน แต่ซื้อมาได้ในราคา 2 พันกว่าๆ บาทจากร้าน Buyspry

ตอนนั้นก็ซื้อมาปุ๊บรอทาง DHL ส่งมาไทยแล้วมาถึงบ้านคิดภาษีบวกค่าอื่นๆ อีก 8 ร้อยกว่าๆ บาท พอจ่ายเรียบร้อยก็เอามาใช้ก็พอใช้ได้ แต่ตัว touchpad กากกกกกกกกกมาก ก็เลยซื้อเมาส์ของ Microsoft Sculpt Comfort ที่เป็นเมาส์บลูทูธมาใช้ ก็เอามาต่อกับตัว tablet แล้วใช้ได้ดีแฮะ เอามาแทนคอมพิวเตอร์ได้ (ในระดับหนึ่ง) ใช้ไปๆ มาๆ ก็ติดขัด ทำอะไรไม่ค่อยได้มากนัก ทีนี้จะทำยังไงดีล่ะ?

ผมก็มองหาเครื่องใหม่ ตอนแรกจะซื้อ ASUS Transformer Book T100 แต่ลองจับแล้วตัวคีย์บอร์ดเล็กกว่าปกติในแนวตั้งครึ่งหนึ่งซึ่งใช้ไม่ถนัดมากกกก ก็มองข้ามรุ่นนี้ไป ลองหาหลายรุ่นแล้วไปๆ มาๆ ซื้อ MacBook Air ขนาด 11 นิ้วมาใช้ ก็เลยสั่งเพิ่มแรมไปเป็น 8GB แล้วรอทางนั้นส่งมาไทย ก็ส่งมาเรียบร้อยปุ๊บ ตั้งค่าเสร็จเรียบร้อยแล้วก็เอามาใช้ก็พบว่า

ใช้และพกพาได้สะดวกมากกกกกกกกกกกกกก

นอกจากพกพาได้สะดวกมาก ก็เป็นเรื่องแบตเตอรี่ที่อึดสุดๆ ทำงานได้ทั้งวันเลยทีนี้ 555

อย่างไรก็ตามมีข้อเสียอยู่สองอย่างคือจอเล็กไป กับไม่มีช่องเสียบ SD Card ต้องเสียบผ่าน USB เอา ตรงนี้อาจจะไม่เหมาะสำหรับบางคนที่ต้องถ่ายรูป หรือโอนไฟล์เข้า SD Card อยู่บ่อยๆ

หลังจากที่ได้แมคมาแล้ว ต่อมาก็ขาดโปรแกรมที่ใช้เป็นประจำอย่าง Microsoft Office ทีนี้จะทำยังไงดีล่ะ ตอนแรกก็เช่าบริการคลาวด์อย่าง Microsoft Azure เพราะเป็น Windows รุ่นที่ใหม่กว่าของ Amazon AWS แล้วเอามาใช้ก็พบว่า

เน็ตที่คณะกากกกกกกก

จบเลยละ 555 ทีนี้ก็เลยล้มแผนนี้ไป ต่อมาผมก็นึกขึ้นได้ว่างานส่วนใหญ่ใช้เป็นภาษาอังกฤษเป็นหลักก็เลยโหลด Microsoft Office 2011 สำหรับแมคลงเครื่องซึ่งหน้าตาก็รับไม่ค่อยได้มากนัก ตอนนี้ก็เลยใช้สลับกันไประหว่าง iWork (Pages, Keynote, Numbers) และ Microsoft Office ขึ้นกับลักษณะงานที่ต้องการ เรื่องปัญหาเอกสารเปิดมาแล้วเพี้ยนนี่ไม่ค่อยเจอมากนัก เพราะใช้เป็นภาษาอังกฤษอยู่แล้วเลยรอดตัวไป

ส่วนโปรแกรมอื่นๆ อย่าง Adobe, Brackets (ไว้เขียนโค้ด), Cyberduck, Firefox, Chrome และอื่นๆ ก็มีอยู่ทั้งสองระบบปฏิบัติการเลยไม่ค่อยมีปัญหามากนัก แต่อาจจะมีปัญหาเวลาที่อยากจะเล่นเกมบนพีซีโคตรจะฮิตซึ่งไม่มีใครทำมาลงบนแมคเลยย อย่างไรก็ตามเรื่องนี้ก็ไม่ค่อยกังวลมากนักเพราะส่วนใหญ่จะเล่นเกมบนมือถือมากกว่านะ 5555

นอกจากนี้โปรแกรมบางตัวที่ทำบนแมคที่โหลดจาก Mac App Store บางตัวก็ดีเกินบรรยายเสียจริงๆ

สิ่งที่ดีสำหรับแมคอีกอย่างหนึ่งที่ไม่อยากจะกล่าวมากนักก็คือ

แมคไม่มี Baidu PC Faster, Baidu Antivirus นะคร้าบบบบบบบ

เอาเข้าจริงโปรแกรมทั้งสองตัวนี้ก็ไม่คิดจะใช้อยู่แล้ว (ฮา) แต่ลองใช้ Search Engine อย่าง Baidu แล้วก็ใช้ได้ดีเลยล่ะ แต่ผมติด Google มากกว่า 555

ขาย iPad ให้ร้านตู้ในมาบุญครองครั้งแรก

วันศุกร์ที่ผ่านมาที่เป็นวันพืชมงคลเราก็เอา iPad 3 Wi-Fi 32GB สีดำ (เครื่องหิ้ว) ไปขายที่ร้านตู้ในห้างมาบุญครองเพราะไม่อยากจะเก็บไว้ให้รกบ้าน และให้ราคาตกไปเรื่อยๆ

โดยก่อนที่จะเอาไปขายก็

  • ใส่อุปกรณ์ที่มีมาให้ลงไปในกล่อง
  • ล้างเครื่อง iPad ให้ข้อมูลทุกอย่างหายไปให้หมด
  • เช็คราคากับทางเว็บไซต์ต่างๆ โดยราคาที่ขายรุ่นนี้เป็นมือสองก็ราวๆ 7 - 8 พันกว่าๆ บาท (หรือน้อยกว่า)

 

เมื่อเตรียมตัวตามสามข้อแล้ว เราก็เอา iPad ไปขายที่ร้านตู้ในห้างมาบุญครอง ซึ่งก็แอบเสียวๆ เหมือนกันว่าจะเอา iPad ไปแกะหรือทำอะไรหรือเปล่า ก็เอาไปขายแล้วทางร้านก็แกะกันรอย (ทั้งตัว) พบว่าสภาพเหมือนใหม่สุดๆ ตีราคาได้ประมาณ 7 พันบาท (อีกร้านที่เอาไปให้ประเมินราคาก็ได้ราวๆ 6 พันกว่าๆ บาท ซึ่งต่ำกว่าอีก) ก็ขายในราคานั้นไป เพราะ iPad 3 ตกรุ่นไปประมาณสองปีแล้ว แถมยังเป็นรุ่น Wi-Fi

ต่อมาทางร้านก็นำ iPad ไปทดสอบว่า battery มีปัญหาหรือเปล่า ซึ่งจะใช้เวลาไม่เกิน 10 นาที ตอนนั้นก็ไปนานเกินแล้วกลัวว่าจะเอาไปทำอะไรหรือเปล่าวะ ก็นั่งรอซักระยะหนึ่ง (จำระยะเวลาไม่ได้) ก็พบว่าไม่ได้มีปัญหาอะไร สุดท้ายก็ขายให้กับทางร้านตู้ได้เรียบร้อย

วิธีการทดสอบ battery ของทางร้านตู้ก็นำ iPad ไปชาร์ตเพื่อดูว่าชาร์ตเร็วผิดปกติหรือไม่ จากนั้นเมื่อชาร์ตแล้วก็ปล่อย iPad ให้ทำงานซักระยะหนึ่งเพื่อดูการลดระดับ battery ว่าลดเร็วจนผิดปกติหรือไม่ ก็ประมาณนี้ครับ

จะซื้อโน้ตบุ๊คซักเครื่อง...รู้จักกับโน้ตบุ๊คแต่ละยี่ห้อ

สำหรับคนที่เข้ามาอ่านในหน้านี้อยากจะบอกว่าเนื้อหาบางส่วนอาจจะไม่ได้อัพเดตนะครับ ถ้าต้องการรู้ข้อมูลของแต่ละยี่ห้อจริงๆ แนะนำให้หาอ่านเว็บอื่น ได้แก่ Notebookspec เป็นต้น

ในปัจจุบันมีโน้ตบุ๊คอยู่หลายยี่ห้อ แต่ละยี่ห้อก็มีหลากหลายรุ่น ในแต่ละรุ่นจะแตกต่างในเรื่องการใช้งาน วัสดุ สเปก (บางยี่ห้ออาจจะต่างไม่มาก) ออฟชั่นต่างๆ เช่นใส่ซิมได้ เป็นต้น ซึ่งอาจจะทำให้บางคนเวียนหัวได้ (ต้องใช้ยาดมยี่ห้อหนึ่ง...ไม่ใช่และ) ในบทความนี้จะพูดถึงความแตกต่างของแต่ละยี่ห้อ แต่ละรุ่นนะครับ

Acer

Acer เป็นยี่ห้อที่ผลิตคอมพิวเตอร์สัญชาติไต้หวัน กับเป็นยี่ห้อที่ได้รับความนิยมยี่ห้อหนึ่งในไทย ซึ่งมีคุณภาพที่พอใช้ได้ อาจจะสู้รุ่นอื่นไม่ได้ แต่ราคาที่ไม่แพงมาก (อาจจะไม่เสมอไปในบางรุ่น) โดยจะมีรุ่นของโน้ตบุ๊คที่ยังผลิตอยู่ซักสามสี่รุ่นได้แก่ Chromebook, Iconia, Aspire, Travelmate

Chromebook คือโน้ตบุ๊คที่ใช้ระบบปฏิบัติการ Chrome OS ของ Google ซึ่งจะเน้นการเข้าเว็บอย่างเดียว ที่ต้องเชื่อมต่อกับอินเตอร์เน็ตตลอดเวลา ถ้าไม่มีอินเตอร์เน็ตอาจจะมีแอปที่สามารถใช้งานได้ แต่ไม่สะดวกแน่ๆ และอินเตอร์เน็ตในไทยก็อ่านะ....

Acer เป็นเจ้าหนึ่งที่ผลิต Chromebook มาตั้งแต่สมัยสองสามปีที่แล้ว ซึ่งเป็นรุ่นหนึ่งที่ได้รับความนิยมอย่างดี เนื่องจากราคาที่ถูก กับประสิทธิภาพดี และมีแบตเตอรี่ที่นาน (เฉพาะรุ่นหลังๆ นะครับ) โดยจะมีไม่กี่รุ่นได้แก่ Acer C700, C710, C720 และ C720P ซึ่งรุ่น Acer C720P จะเป็นรุ่นเดียวที่เป็นจอสัมผัสในปัจจุบัน

โน้ตบุ๊ครุ่นใหม่ที่ออกมาไม่นาน (จริงๆ นานแล้ว) ก็เป็นรุ่น Acer C720 กับ C720P ซึ่งมีแบตเตอรี่ที่นาน มีประสิทธิภาพดีกว่าโน้ตบุ๊ค Chromebook ของยี่ห้ออื่นๆ

ปล. Chromebook ยังไม่มีขายในไทย (ฝากคนหิ้วได้นะ 55)

Iconia

จริงๆ รุ่น Iconia เป็นรุ่นที่ทำเกี่ยวกับ tablet แต่บางรุ่นก็ทำมาแบบ hybrid คือจะเป็น tablet หรือ โน้ตบุ๊ค ก็ได้ ในปัจจุบันมี Windows tablet ขนาดหน้าจอ 8 นิ้วที่วางขายในไทยได้ซักระยะหนึ่งแล้ว เป็นรุ่น Iconia W4 (รุ่นเดียวกับในวิดีโอที่แปะด้านบน) ซึ่งเป็น Windows tablet ที่สามารถใส่ซิมต่อ 3G ได้เลย กับแถมโปรแกรม Microsoft Office 2013 Home and Student ให้เลย ซึ่งเป็น tablet ที่สะดวก อย่างไรก็ตามหน้าจออาจจะเล็กเกินไป ทำให้ใช้งานไม่สะดวกนัก ถ้าใช้จริงจังอาจจะต้องต่อหน้าจอแล้วใช่คีย์บอร์ดกับเมาส์บลูทูธจะสะดวกกว่า

Aspire

Acer Aspire เป็นโน้ตบุ๊ครุ่นหนึ่งที่ออกแบบมาเพื่อใช้งานได้หลากหลาย ตั้งแต่งานทั่วๆ ไป ไปจนถึงเล่นเกม ทำงานกราฟฟิก

โน้ตบุ๊ค Acer Aspire เป็นรุ่นที่นิยมอีกรุ่นหนึ่งโดยมีราคาที่หลากหลายมากตั้งแต่หมื่นต้นๆ จนไปถึง 5 - 6 หมื่นบาท ซึ่งมีตั้งแต่โน้ตบุ๊คปกติ ไปจนถึง Ultrabook และ tablet อย่าง Acer Aspire P3 (ซึ่งดูแล้วหนักเกินไปสำหรับ tablet แต่ก็เบามากถ้าเทียบกับโน้ตบุ๊คด้วยกัน)

คุณภาพจริงๆ อยู่ในระดับที่ใช้ได้ครับ ถ้าเลือกรุ่นที่ราคาสูงหน่อยก็ OK ขึ้นไปอีก

Travelmate

Acer Travelmate เป็นโน้ตบุ๊คที่ใช้ในธุรกิจซึ่งไม่ค่อยพบตามร้านค้าทั่วไปนัก (จริงๆ สมัยก่อนมีวางขายตามร้านทั่วไป แต่ก็ไม่เข้าใจว่าทำไมถึงไม่ขาย)

Apple

Apple เป็นบริษัทที่ทำอุปกรณ์หลากหลาย iPod, iPhone, iPad, Macbook, iMac, Mac Mini, Mac Pro เป็นต้น

โน้ตบุ๊คของ Apple จะใช้ระบบปฏิบัติการ Mac OS X ที่น่าจะรู้จักกันอยู่แล้ว (อาจจะไม่เขียนรายละเอียดมาก) ซึ่งมีอยู่สองรุ่นหลักๆ ก็คือ Macbook Air กับ Macbook Pro with Retina Display

Macbook Air

Macbook Air เป็นโน้ตบุ๊คบางเบาที่มีอยู่สองขนาดหน้าจอ (ในปัจจุบัน) คือ 11 นิ้วกับ 13 นิ้ว ซึ่งมีข่าวลือว่าอาจจะทำรุ่น Retina Display 12 นิ้ว ซึ่งมีเสปกที่ดีกว่า Ultrabook ที่แพงกว่ารุ่นอื่น (เพราะรุ่นอื่นจะใช้ CPU Core i5 4200U ซึ่งมีการ์ดจอที่แย่กว่าการ์ดจอที่อยู่ใน CPU Core i5 4250U ใช้ใน Macbook Air)

Macbook Pro with Retina Display

Macbook Pro with Retina Display เป็นโน้ตบุ๊คที่มีหน้าจอความละเอียดสูง กับบางเบา (บางไม่เท่าไรนัก แต่บางกว่ายี่ห้ออื่น) ซึ่งมีสองขนาดหน้าจอ (ในปัจจุบัน) คือ 13 นิ้วและ 15 นิ้ว ซึ่งมีข้อเสียคล้ายๆ กันคืออัพเกรตไม่ค่อยจะได้

ข้อดีของ Macbook Air และ Pro with Retina Display ก็ใช้ SSD แบบ PCI Express ที่มีความในการการอ่าน/เขียนที่สูงมาก กับระบบปฏิบัติการ Mac OS X เอง และอื่นๆ

ข้อเสียของ Macbook Air และ Pro with Retina Display จริงๆ ก็เป็นเรื่องที่อัพเกรตไม่ค่อยจะได้ เพราะส่วนใหญ่ก็บัดกรีบนเมนบอร์ด เพราะฉะนั้นก่อนซื้อก็คิดดีๆ ก่อนนะครับ กับสเปกทีอาจจะต่ำกว่าคู่แข่งในราคาที่พอๆ กัน (ปล. ตอนนี้ราคา Ultrabook ใกล้เคียงหรือแพงกว่า Macbook Air ไปแล้ว)

ASUS

ASUS เป็นยี่ห้อที่ผลิตโน้ตบุ๊คสัญชาติไต้หวัน ซึ่งผลิตตั้งแต่อุปกรณ์คอมพิวเตอร์ router ไปจนถึงโน้ตบุ๊ค คอมตั้งโต๊ะ และอื่นๆ อีกเยอะ (ประมาณว่าผลิตหลายอย่างมาก) ซึ่งโน้ตบุ๊คเองก็มีหลายรุ่นตั้งแต่ Transformer Book, Vivobook, Zenbook, Taichi, โน้ตบุ๊คปกติก็มี N, X, K และโน้ตบุ๊คสำหรับเล่นเกมก็ G series ข้อดีของโน้ตบุ๊คยี่ห้อนี้คือประกันจะนานกว่ารุ่นอื่น (ถ้าเทียบกับรุ่นที่ใช้ตามบ้านนะ) ส่วนข้อเสียไม่ค่อยทราบเพราะไม่ได้ใช้รุ่นนี้ครับ

Transformer Book

จริงๆ รุ่น Transformer มาจาก tablet บน Android มาก่อนค่อยทำ Transformer Book ทีหลังซึ่งเป็น Windows tablet อีกรุ่นหนึ่ง ในปัจจุบันก็จะมีอยู่ซักสองสามรุ่นได้แก่

Transformer Book T100 ที่ใช้ CPU Intel Atom ที่สามารถใช้ได้อย่างยาวนาน แต่สเปกจะไม่แรงมากเท่า CPU ของ Intel รุ่นอื่น นอกจากนี้ยังให้ Windows กับ Office 2013 Home and Student ของแท้มาให้ เท่าที่ขายในไทยจะมีสองรุ่นคือรุ่นที่ให้ความจุ 64GB (ในตัว tablet) กับอีกรุ่นคือ 32GB (ในตัว tablet) และ 500GB (ในตัว Keyboard)

Transformer Book Trio ที่มีทั้ง Android และ Windows ในเครื่องเดียว

Transformer Book TX300CA ซึ่งเก่าสุดเมื่อเทียบกับสองรุ่นแรก

Vivobook

Vivobook เป็นโน้ตบุ๊คจอสัมผัสของ ASUS รุ่นหนึ่ง ซึงมีหลายขนาดกับหลายสเปก กับขนาดหน้าจอที่หลากหลายตั้งแต่ 11, 13, 14 และ 15 นิ้ว และแถมระบบปฏิบัติการ Windows 8 มาให้ (ไม่แน่ใจว่าให้ทุกรุ่นหรือไม่)

เท่าที่ทราบในปัจจุบันยังไม่มีการอัพเดตเป็น CPU รุ่นใหม่ และระยะเวลาการใช้งานเมื่อเปิด Wi-Fi ไม่นานนัก (แต่ถ้าเปิดทิ้งไว้ก็สามารถใช้ได้นานพอสมควร)

Zenbook

Zenbook เป็นโน้ตบุ๊คบางเบารุ่นหนึ่งของ ASUS ที่มีขนาดหน้าจอ 13.3 นิ้วที่มีหน้าจอความละเอียด Full HD เหมาะกับคนที่พกพาไปไหนบ่อยๆ และยังต้องการประสิทธิภาพที่ยังสูงอยู่

อย่างไรก็ตามราคา ASUS Zenbook ที่ขายในไทย อาจจะสูงเกินเอื้อมสำหรับหลายคนเพราะเริ่มต้นก็อยู่ที่ 49,900 บาท (อ้างอิงตาม ASUS Product Guide May - June 2014) ซึ่งเดิมทีโน้ตบุ๊ครุ่นนี้ราคาไม่แพงเกินเอื้อมไปมากนัก ประมาณ 3 หมื่นต้นๆ แต่ก็ไม่เข้าใจว่าทำไมขายแต่รุ่นที่ใช้ CPU Core i7 = ="

Taichi

Taichi เป็นโน้ตบุ๊คที่มีสองขนาดให้เลือกคือ 11 และ 13 นิ้ว ซึ่งจะมีสองหน้าจอคือหน้าจอที่เป็นหน้าจอสัมผัสกับไม่สัมผัส ที่มีน้ำหนักที่เบามาก แต่สเปกก็รุ่นเก่าแล้ว ยังไม่มีการอัพเดตเป็นรุ่นใหม่

โน้ตบุ๊คทั่วๆ ไป N,X,K series

โน้ตบุ๊คทั่วๆ ไปก็ขึ้นอยู่กับการใช้งานของเรามากกว่าอย่างเช่น N series ก็จะออกแนวเน้นไปทาง Multimedia เล่นเกม มากกว่า ส่วน X,K series ก็จะเป็นโน้ตบุ๊คที่ใช้งานทั่วๆ ไปครับ

โน้ตบุ๊คสำหรับเล่นเกม G series

โน้ตบุ๊คสำหรับเล่นเกม ก็จะมีอยู่สองขนาดหน้าจอได้แก่ 15 และ 17 นิ้ว ซึ่งออกแบบมาเพื่อเล่นเกมโดยเฉพาะ สเปกก็จะแรงพอสมควร

ในสมัยก่อนโน้ตบุ๊ครุ่นนี้จะมีราคาที่แพงมาก แต่สมัยนี้ราคาถูกลงมาแล้ว

Dell

Dell เป็นบริษัทที่ผลิตคอมพิวเตอร์สัญชาติอเมริกาบริษัทหนึ่งที่ทำตั้งแต่คอมตั้งโต๊ะ โน้ตบุ๊ค server เป็นต้น

ข้อดีของโน้ตบุ๊คของบริษัทนี้ที่ทำให้โน้ตบุ๊คยี่ห้อนี้ได้รับความนิยมคือเรื่องการรับประกันที่รับประกันแบบ On-site service (ถ้าโน้ตบุ๊คมีปัญหาก็ซ่อมถึงที่ได้เลย)

สิ่งที่แตกต่างระหว่างโน้ตบุ๊ค Inspiron, Vostro และ Latitude ก็คือการรับประกัน ราคา วัสดุ เป็นต้น แต่จุดที่น่าจะคล้ายๆ กันก็คือเรื่องของสเปกมากกว่า

Inspiron

โน้ตบุ๊ครุ่นนี้มีหลายราคา ตั้งแต่ราคาหมื่นต้นๆ ไปจนถึงสี่ห้าหมื่น กับหลายขนาดหน้าจอตั้งแต่ 11 นิ้วไปจนถึง 17 นิ้ว และมีหลายเกรด

Vostro

Dell Vostro เป็นโน้ตบุ๊คที่ออกแบบมาเพื่อธุรกิจขนาดเล็ก (จริงๆ คนทั่วไปก็ใช้ได้) ราคาเริ่มต้นอยู่ที่ประมาณสองหมื่นต้นๆ ขึ้นไป

Latitude

ข้อดีของโน้ตบุ๊ครุ่นนี้คือเวลาที่สั่งซื้อมาจะได้รับประกันสามปี และทุกเครื่องจะมีระบบปฏิบัติการ Windows แท้ติดตั้งให้เลย

Precision

Dell Precision เป็นโน้ตบุ๊คที่ออกแบบมาเพื่อทำงานด้านกราฟฟิก การคำนวณทางคณิตศาสตร์ งานวิจัยทางวิทยาศาสตร์ เป็นต้น ซึ่งมีสเปกที่สูงเพื่อรองรับงานที่ทำอยู่ กับมีขนาดหน้าจอตั้งแต่ 14 นิ้วไปจนถึง 17 นิ้ว และราคาก็มีเริ่มตั้งแต่ราวๆ 5 หมื่นบาทขึ้นไป (กรณีสั่งเครื่องเดียว)

แต่เดิมโน้ตบุ๊ครุ่นนี้จะหนักและหนามาก จนกระทั่งโน้ตบุ๊ครุ่น Dell Precision M3800 (ในวิดีโอ) ที่ออกมาซักระยะหนึ่ง ซึ่งเป็นโน้ตบุ๊คที่บางและเบากว่ารุ่นก่อนๆ

Chromebook

Dell พึ่งมาผลิต Chromebook ที่มีขนาดหน้าจอ 11 นิ้ว และวางขายในสหรัฐได้ไม่นาน โน้ตบุ๊ครุ่นนี้ใช้ระบบปฏิบัติการ Chrome OS ของ Google ซึ่งมีแบตเตอรี่ที่ใช้ได้อย่างยาวนาน และมีราคาที่ไม่แพงมาก อย่างไรก็ตามโน้ตบุ๊ครุ่นนี้ก็ไม่มีขายในไทย = ="

HP

HP เป็นบริษัทที่ผลิตคอมพิวเตอร์สัญชาติอเมริกา ซึ่งเคยมีปัญหาเรื่องวิธีการแก้ปัญหาโน้ตบุ๊คที่มีปัญหาเกี่ยวกับความร้อนของการ์ดจอ (Heat Cycle) ของการ์ดจอรุ่น NVIDIA 8400M GS แต่ตอนนี้คิดว่าน่าจะดีขึ้นแล้ว

โน้ตบุ๊คยี่ห้อ HP ผลิตออกมา 5 รุ่นได้แก่ HP Compaq, Pavilion, Probook, Elitebook และ ZBook

Compaq Presario

เดิม Compaq เป็นบริษัทที่ผลิตคอมพิวเตอร์ก่อนที่บริษัท Compaq จะถูกบริษัท HP ซื้อ

HP Compaq Presario เป็นโน้ตบุ๊ครุ่นหนึ่งของ HP ที่ออกแบบมาเพื่อใช้งานทั่วไป ซึ่งเป็นรุ่นที่ราคาไม่แพงมากนัก ซึ่งไม่ทราบว่ายังมีขายอยู่หรือไม่

Pavilion

HP Pavilion เป็นโน้ตบุ๊ครุ่นหนึ่งที่ผลิตมาเพื่อใช้งานทั่วไป จนถึงเล่นเกม ทำงานกราฟฟิก ซึ่งมีราคา ขนาดหน้าจอ วัสดุ คุณภาพที่หลากหลาย

Probook

HP Probook เป็นโน้ตบุ๊ครุ่นที่ผลิตมาเพื่อใช้งานในธุรกิจขนาดเล็ก (แต่จริงๆ ใครๆ ก็ใช้งานได้นะครับ) มีหลากหลายสเปกตั้งแต่โน้ตบุ๊คเบาบาง จนกระทั่งโน้ตบุ๊คที่ออกแบบมาเพื่อทำงานกราฟฟิก

Elitebook

HP Elitebook เป็นโน้ตบุ๊คที่ผลิตมาเพื่อใช้ในองค์กร ซึ่งคล้ายๆ กับ Dell Latitude ที่มักจะให้องค์กรสั่งเครื่องมากกว่า

โน้ตบุ๊ครุ่นนี้จะมีหน้าจอตั้งแต่ 11 นิ้วไปจนถึง 17 นิ้ว (หรือ 15 นิ้วก็ไม่ทราบนะ) มีอยู่สองแบบคือ tablet (เช่น HP Elitebook Revolve) และ notebook

ZBook

HP ZBook เป็นโน้ตบุ๊คที่ออกแบบมาเพื่องานกราฟฟิก งานคำนวณทางคณิตศาสตร์ วิทยาศาสตร์ เป็นต้น

เดิมโน้ตบุ๊ครุ่น HP ZBook จะอยู่ในรุ่น HP Elitebook ที่จะเป็นรหัสสี่หลักที่ลงท้ายด้วย w

HP ZBook มีขนาดหน้าจอที่หลากหลายตั้งแต่ 14 นิ้วซึ่งจะเป็น Ultrabook, 15 นิ้ว และ 17 นิ้ว และราคาเริ่มต้นอยู่ที่ 5 หมื่นบาทขึ้นไป

Chromebook

HP ทำ Chromebook และวางขายออกมาพอสมควรแล้ว ซึ่งมีขนาดหน้าจออยู่สองขนาดคือ 11 นิ้วที่ใช้ CPU แบบเดียวกับมือถือ และหน้าจอ 14 นิ้วที่ใช้ CPU แบบเดียวกันกับคอมทั่วๆ ไป อย่างไรก็ตามก็ไม่มีขายในไทยอยู่ดี

Lenovo

Lenovo เป็นบริษัทที่ผลิตคอมพิวเตอร์สัญชาติจีนบริษัทหนึ่ง

ข้อดีของโน้ตบุ๊คยี่ห้อนี้ที่ได้รับความนิยมคือ ราคากับสเปคที่เหมาะสมซึ่งทำให้ได้รับความนิยม

ในปัจจุบัน Lenovo ผลิตโน้ตบุ๊คออกมาสองรุ่นได้แก่ IdeaPad และ ThinkPad

IdeaPad

IdeaPad เป็นโน้ตบุ๊คที่ออกแบบมาเพื่อผู้ใช้ตามบ้าน เล่นเกม หรืออื่นๆ มีหลากหลายราคา สเปค วัสดุ ตั้งแต่ราคาหมื่นต้นๆ ไปจนถึงหลายหมื่น

ThinkPad

ThinkPad เป็นโน้ตบุ๊คที่ออกแบบมาเพื่อองค์กรธุรกิจ ซึ่งแต่เดิมเป็นของ IBM ต่อมาบริษัท Lenovo ซื้อเอาไปทำต่อ

คุณภาพของโน้ตบุ๊ครุ่นนี้เดิมมีคุณภาพดีมาก ต่อมาคุณภาพ กับการรับประกันที่แย่ลง ทำให้หลายๆ คนเปลี่ยนใจไปซื้อโน้ตบุ๊ครุ่นอื่น ได้แก่ Macbook Pro, Dell Latitude เป็นต้น

ThinkPad มีสองรุ่นย่อยได้แก่ ThinkPad Edge ที่ออกแบบมาเพื่อนักธุรกิจรุ่นใหม่ (หรือเปล่า) และ ThinkPad (ที่มีมาตั้งแต่สมัย IBM) ซึ่งมีข้อแตกต่างเรื่องความทนทาน ลักษณะรูปร่างภายนอก ราคา เป็นต้น ส่วนใหญ่ถ้าจะแนะนำจริงๆ ก็แนะนำ ThinkPad มากกว่า

Microsoft

tablet ของ Microsoft ที่ใช้ CPU แบบเดียวกับโน้ตบุ๊คมีอยู่สองรุ่น Surface Pro กับ Surface Pro 2 (ซึ่งยังไม่มีวางขายในไทย) โดยมีแนวคิดประมาณว่าโน้ตบุ๊คที่อยู่ในรูปแบบ tablet ข้อดีอย่างหนึ่งของ tablet รุ่นนี้คือจะได้รับการอัพเดตจาก Microsoft แน่ๆ ซึ่งคล้ายๆ กับ tablet ของ Google อย่าง Nexus

ข้อดีของ tablet รุ่นนี้คือพกพาได้สะดวกกว่า Ultrabook, มีปากกามาให้เหมาะกับการเขียน และมีอุปกรณ์เสริมพอสมควร นอกจากอุปกรณ์เสริมที่มีขายแยกแล้ว ยังสามารถซื้ออุปกรณ์ที่มีอยู่ในท้องตลาดอยู่แล้วมาใช้ได้

MSI

MSI เป็นบริษัทสัญชาติไต้หวันที่ทำอุปกรณ์คอมพิวเตอร์หลากหลาย ตั้งแต่เมนบอร์ด การ์ดจอ คอมพิวเตอร์ตั้งโต๊ะ โน้ตบุ๊ค เป็นต้น

Samsung

Samsung เป็นบริษัทสัญชาติเกาหลีที่ผลิตเครื่องใช้ไฟฟ้าที่หลากหลายมาก (จนจำไม่ได้)

ในปัจจุบัน โน้ตบุ๊คของ Samsung จะแบ่งออกมาเป็นสองรุ่นคือ ATIV Book กับ ATIV Tab ที่เป็น Windows tablet

เรื่องคุณภาพของโน้ตบุ๊คนี่บอกไม่ได้เพราะไม่เคยใช้โน้ตบุ๊ครุ่นนี้ครับ

Sony

Sony เป็นบริษัทสัญชาติญี่ปุ่นที่ผลิตเครื่องใช้ไฟฟ้าหลากหลายมาก

ในปัจจุบัน โน้ตบุ๊คของ Sony (Sony VAIO) ได้ถูกขายให้กับ JIP (Japan Industrial Partners) แล้ว ซึ่งตอนนี้ไม่น่าจะมีขายในบางร้านในประเทศไทยแล้ว

Toshiba

Toshiba เป็นบริษัทสัญชาติญี่ปุ่นเช่นเดียวกันกับ Sony ที่ผลิตเครื่องใช้ไฟฟ้ามากมายอีกเช่นกัน

ในปัจจุบัน โน้ตบุ๊คของ Toshiba มี 4 รุ่นได้แก่ Satellite, Portege, Qosmio และ Tecra ซึ่งโน้ตบุ๊คของ Toshiba มีข้อเสียเท่าที่เคยอ่านผ่านๆ คือโน้ตบุ๊คมีสเปกที่ต่ำกว่ารุ่นอื่นในราคาที่พอๆ กัน

Satellite

โน้ตบุ๊ค Satellite เป็นโน้ตบุ๊คที่ออกแบบมาเพื่อให้ใช้งานได้หลากหลาย ตั้งแต่งานทั่วๆ ไป ไปจนถึงงานกราฟฟิก หรืออื่นๆ มีราคาที่หลากหลายเช่นกัน

Portege

โน้ตบุ๊ค Portege เป็นโน้ตบุ๊คที่ออกแบบมาเพื่อใช้งานธุรกิจ (จริงๆ คนทั่วไปก็ใช้ได้นะครับ) ที่มีหลากรุ่นอีกเช่นกัน โดยจะมีโน้ตบุ๊คปกติ ไปจนถึงโน้ตบุ๊คบางเบาอย่างเช่น R และ Z Series ซึ่ง Z Series เป็นโน้ตบุ๊คที่บางมากและเบามากกว่า Macbook Air 13 นิ้ว แต่มีราคาที่แพงกว่าเช่นกัน และ R Series เป็นโน้ตบุ๊คบางเบาที่ยังคงเน้นความแรงของ CPU ที่ใช้ CPU โน้ตบุ๊คปกติ ไม่ใช่แบบประหยัดพลังงาน

Qosmio

โน้ตบุ๊ค Qosmio เป็นโน้ตบุ๊คที่ออกแบบมาเพื่องาน Multimedia กับเล่นเกม และทำงานกราฟฟิก เป็นต้น

โน้ตบุ๊ครุ่นนี้เป็นโน้ตบุ๊คที่เน้นมาเพื่อประสิทธิภาพที่สูง อย่างไรก็ตามราคาของโน้ตบุ๊ครุ่นนี้มีราคาที่สูงกว่ารุ่นอื่นที่มีสเปคใกล้เคียงกัน

Tecra

จะซื้อคอม (โน้ตบุ๊ค) ซักเครื่อง...ทำไงดี

ปกติเวลาจะที่เราซื้อโน้ตบุ๊คซักเครื่องหนึ่ง ผมจะถามไปประมาณสามคำถามได้แก่ จะเอาไปทำอะไร, มีงบประมาณเท่าไร และต้องการพกพาบ่อยไหม ตอนนี้ผมจะเริ่มที่คำถามแรกก่อนเลยครับ

จะเอาไปทำอะไร?

คำถามแรกที่ถามนี้ จะถามเพื่อที่จะรู้ว่าเอาโน้ตบุ๊คไปทำอะไร เช่น เล่นเกม, ทำงาน Office, ทำงานกราฟฟิก เป็นต้น แล้วเอาไปพิจารณาสเปกที่เหมาะสมสำหรับความต้องการของคนที่ถามครับ โดยการใช้งานที่พบได้บ่อยคือ เอาไปทำงานทั่วๆ ไป เล่นเกมนิดหน่อย

มีงบประมาณเท่าไร?

คำถามนี้สำคัญเลยนะครับ ถ้ามีงบประมาณไม่พอนี่ ซื้อโน้ตบุ๊คไม่ได้เลยนะครับ 5555 (เอาจริงๆ สิฟร่ะ) งั้นเอาจริงๆ เลยล่ะกันครับ

ที่ถามว่ามีงบประมาณเท่าไร คำถามนี้จะพูดถึงงบประมาณที่จะซื้อโนตบุ๊คโดยที่ราคาไม่สูงเกินกว่างบประมาณที่กำหนดมาให้ครับ (จริงๆ อาจจะสูงกว่าได้นิดหน่อย แต่ไม่สูงเกินไปนะแหล่ะครับ) โดยที่พบบ่อยก็ไม่เกิน 2-3 หมื่นบาท

ต้องการพกพาบ่อยไหม?

คำถามสุดท้ายจะพูดถึงน้ำหนัก ความบางของโน้ตบุ๊คครับ คำถามนี้จะสำคัญสำหรับคนที่พกพาโน้ตบุ๊คเป็นประจำ เพราะถ้าได้โน้ตบุ๊คที่เสปกดีก็จริง แต่มีน้ำหนักที่มากและหนาจนเกินไป ทำให้พกพาไม่สะดวก จนถึงขึ้นไม่พกพาก็ได้ครับ ดังนั้นโน้ตบุ๊คที่คิดว่าเหมาะสำหรับคนที่เดินทางเป็นประจำเลยน่าจะเป็น Ultrabook ครับ

Ultrabook คือโน้ตบุ๊คบางเบาในรูปแบบหนึ่งที่กำหนดโดย Intel ครับ ส่วนข้อกำหนดที่จะบอกว่าโน้ตบุ๊ครุ่นนี้เป็นหรือไม่เป็น ก็ดูได้จาก Wikipedia หน้า Ultrabook ตรงส่วนของ Specifications ได้เลยครับ

หลังจากที่ตอบคำถามแล้ว

ก็หาข้อมูลเพิ่มเติมก่อนที่จะเลือกโน้ตบุ๊ค โดยจะแนะนำที่ผมหาข้อมูลนะครับ

ผมมักจะหาข้อมูลบ่อยๆ ก็ค้นหาได้ใน Google กับเปิดเว็บ โดยส่วนใหญ่ที่เปิดหาก็เป็นเว็บ Notebookspec (เว็บนี้ไว้เช็คราคากับดูรีวิว), Notebookcheck (ดูรีวิวและดูความแรงการ์ดจอกับ CPU), YouTube (ค้นหาพวกรีวิวโน้ตบุ๊ค), Engadget, The Verge

นอกจากจะหาข้อมูลแล้ว ควรที่จะหาข้อมูลด้วยว่าโน้ตบุ๊ครุ่นนั้นมีปัญหาหรือไม่ การประกันเป็นอย่างไร (บางแบรนด์เช่น Dell จะซ่อมถึงที่ได้เลย) อัพเกรตได้ไหม (จริงๆ อาจจะไม่ต้องซีเรียสนักก็ได้ เพราะบางรุ่นก็อัพเกรตแทบจะไม่ได้อยู่แล้ว) ให้ระบบปฏิบัติการอะไรมา (โน้ตบุ๊คบางรุ่นอาจจะให้ระบบปฏิบัติการ Windows, Mac OS X มา แต่รุ่นอื่นๆ มักจะให้เป็น DOS หรือ Linux (Ubuntu) ครับ)

จริงๆ ถึงแม้ไม่มีระบบปฏิบัติการมาให้ก็สามารถซื้อ Windows แท้มาลงได้ครับ (อาจจะลง Mac OS X ในโน้ตบุ๊คทั่วไปได้ แต่ไม่รู้ว่าลงแล้วจะเป็นอย่างไรนะครับ)

เรื่องอัพเกรตโน้ตบุ๊ค บางรุ่นต้องอัพเกรตตอนที่จะซื้อเลยเช่น Macbook Air, Macbook Pro with Retina Display เป็นต้น ดังนั้นคิดดีๆ ก่อนที่จะซื้อนะครับ

เมื่อหาข้อมูลเสร็จแล้วก็ตัดสินใจได้แล้วว่าจะซื้อโน้ตบุ๊ครุ่นไหน พอตัดสินใจเสร็จแล้วก็ไปลองจับโน้ตบุ๊คในร้านก่อนว่าถูกใจเราหรือเปล่า ถ้าถูกใจแล้วก็ซื้อโน้ตบุ๊คครับ

แต่ก่อนที่จะซื้อโน้ตบุ๊ค ควรที่จะเช็คสภาพเครื่องก่อนว่ามีปัญหาหรือเปล่า (เช่นจอมี Dead Pixcel) มีรอยต่างๆ ที่ตัวโน้ตบุ๊คหรือไม่ อุปกรณ์ที่มีมาให้ครบหรือเปล่าประมาณนี้ พอเช็คทั้งหมดเสร็จแล้วจ่ายเงินได้เลยครับ

หลังจากจ่ายเงินแล้วก็อาจจะให้ช่างลงระบบปฏิบัติการกับโปรแกรมต่างๆ ให้ (แต่จริงๆ ก็ค่อยมาลงที่บ้านก็ได้ เพราะระบบปฏิบัติการสมัยนี้ที่น่าจะติดตั้งง่ายกว่ารุ่นเก่าๆ แล้ว)

การเลือก Windows แท้

หลังจากที่ซื้อเสร็จแล้วก็ซื้อ Windows ของแท้มาด้วยถ้าโน้ตบุ๊คเครื่องนั้นยังไม่ได้ลงระบบปฏิบัติการเลย โดยจะมีแบบต่างๆ ให้เลือกได้แก่ OEM (อันนี้ซื้อติดกับเครื่อง ย้ายไม่ได้), FPP (เป็นแบบกล่องดีๆ แหละครับ อันนี้ย้ายเครื่องได้)

นอกจากเลือกว่าจะเอาแบบ OEM แล้วยังต้องเลือกรุ่นของ Windows โดย Windows รุ่นใหม่ล่าสุดก็เป็นรุ่น Windows 8.1 ซึ่งจะมีสองแบบอีกคือ SL (Single Language - เลือกได้ภาษาเดียว) กับ Pro โดยมีจุดที่แตกต่างของทั้งสองแบบคือพวกโปรแกรมบางตัวเช่น Hyper-V, การเลือกภาษาที่รุ่น Pro จะเลือกภาษาทีหลังได้ และราคาทีแตกต่าง (แน่นอนสิฟร่ะ)

สำหรับรายละเอียดพวกราคาของ Windows สามารถเช็คได้ในเว็บทั่วๆ ไปครับ

หลังจากที่ซื้อและ (อาจจะ) ติดตั้งโปรแกรมแล้ว

พอทำทุกอย่างเสร็จแล้วก็เอากลับบ้านได้เลยครับ เมื่อกลับถึงบ้านแล้วถ้าโน้ตบุ๊คเกิดปัญหาขึ้นมาก็เปลี่ยนได้ภายใน 7 วัน (ขึ้นกับร้านนะครับ) ดังนั้นควรที่จะระวังในช่วง 7 วันหลังซื้อโน้ตบุ๊คนะครับ ^^
