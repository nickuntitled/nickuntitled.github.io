#6 - ดูทวิตโดยไม่ล็อคอิน Twitter ด้วย Nitter

2023-09-04

Nitter คือเครื่องมือสำหรับทำเว็บเซิร์ฟเวอร์ที่อนุญาตให้ผู้ใช้สามารถเข้าไปดูทวิตใน Twitter (หรือ X) ที่เน้นเรื่องความเป็นส่วนตัว และประสิทธิภาพของการใช้งานเว็บไซต์ โดยโครงการนี้ได้รับแรงบันดาลใจจากโครงการ Invidious

จุดที่แตกต่างจาก Twitter

จุดที่ Nitter แตกต่างกับ Twitter ตามที่เอาข้อมูลมาจากทาง GitHub ของเครื่องมือนี้มีรายละเอียดตามด้านล่างนี้

  • ไม่มีจาวาสคริป และไม่มีโฆษณา
  • ข้อมูลที่เรียกเข้าไปดูทางทวิตจะถูกเก็บในเซิร์ฟเวอร์
  • ตัวเว็บไซต์ป้องกันทางทวิตเตอร์ติดตามเราผ่านทาง IP หรือผ่านทางจาวาสคริป
  • ใช้ Twitter’s unofficial API (ไม่มีลิมิต และไม่จำเป็นต้องมีแอคเค้าที่เป็นนักพัฒนา)
  • ประหยัดทรัพยากร
  • มี RSS Feed
  • แต่งธีมได้
  • ตัวหน้าเว็บรองรับการเปิดบนโทรศัพท์มือถือ
  • มีลิขสิทธิ์แบบ AGPLv3

เหตุผลที่ใช้ Nitter

ต่อมา แล้วทำไมเราถึงไม่เข้าไปผ่านทาง Twitter ปกติ? สาเหตุก็เนื่องมาจากผู้ใช้ไม่สามารถเข้าไปใช้ Twitter ได้โดยที่ไม่เปิดใช้งานจาวาสคริป นอกจากนี้กรณีที่ยังใช้งานตัวบล็อกโฆษณา และใช้งานผ่าน VPN แล้ว ระบบยังติดตามการใช้งานของเราผ่านทางการติดตาม Browser’s fingerprint

ดังนั้นแล้ว ตัวเว็บ Nitter ที่อนุญาตให้เราเข้าไปดูทวิตได้โดยไม่มีจาวาสคริป ทำให้ตัวเว็บไม่สามารถติดตามการใช้งานของเราเพื่อกำหนดว่าจะแสดงโฆษณาแบบไหนได้อีก

วิธีใช้ Nitter เบื้องต้น

ผู้ใช้สามารถดูรายการเซิร์ฟเวอร์ที่เปิดอยู่ได้ทางไหน? อันนี้ทำได้ไม่ยาก เราสามารถเข้าไปดูรายการที่มีอยู่ได้จากหน้าเว็บของ GitHub ที่ทางชุมชนสร้างขึ้นครับ

เมื่อดูรายการในนั้นแล้วเลือกเซิร์ฟเวอร์ใดเซิร์ฟเวอร์หนึ่ง ผู้ใช้จะพบหน้าตาปรากฏตามด้านล่างนี้ (บทความนี้ใช้เซิร์ฟเวอร์ nitter.net)

การค้นหาไอดี

ผู้ใช้สามารถค้นหาไอดีของทวิตที่ต้องการได้โดย

วิธีแรก พิมพ์ที่ช่อง Enter username แล้วกดปุ่มที่รูปแว่นขยาย (หรือกดปุ่ม Enter) เมื่อกดปุ่มแล้วระบบจะปรากฏผลการค้นหาตามด้านล่างนี้

วิธีที่สอง กดปุ่มที่แว่นขยายมุมขวาบน แล้วระบบจะปรากฏหน้าจอค้นหาให้ผู้ใช้พิมพ์ไอดีทวิต หรือทวีตที่ต้องการค้นหาได้ครับ

บทความนี้ เราใช้ไอดี nicknznick (ผู้เขียน) ในการค้นหา ระบบจะปรากฏผลการค้นหาตามข้างบน เมื่อปรากฏผลการค้นหาแล้ว ผู้อ่านเข้าไปกดที่ไอดีของทวิตเตอร์ที่ต้องการตามภาพได้เลย เมื่อกดปุ่มแล้ว ระบบจะแสดงข้อมูลของไอดีนั้น รวมทวีตที่ไอดีนั้นได้ทวีตเข้าไปใน Twitter

การค้นหาทวีต

ผู้ใช้สามารถค้นหาทวีตได้โดยการพิมพ์ที่ช่อง Enter username ที่หน้าแรกของเว็บ หรือกดปุ่มที่มุมขวาบน แล้วพิมพ์ทวีตที่ต้องการค้นหาได้ จากนั้นกดปุ่ม Enter หรือกดปุ่มแว่นขยาย เมื่อกดแล้ระบบจะแสดงทวีตที่ค้นหาตามด้านล่างนี้ (ในตัวอย่างใช้คำว่า failarmy)

จากที่แนะนำการใช้งานเบื้องต้น ผู้ใช้สามารถ

  • ค้นหาทวีตที่โพสไปแล้ว หรือค้นหาไอดีทวิตที่ต้องการเพื่อเข้าไปดูทวีตได้โดยไม่จำเป็นต้องล็อคอินเข้าไปใน Twitter เลย
  • อีกอย่างในกรณีที่ถูกไอดีนั้นบล็อกไปไม่ว่าจะมาจากเหตุผลอะไรก็ตาม ผู้ใช้สามารถดูทวิตไอดีนั้นได้โดยเข้าใช้งานผ่านเซิร์ฟเวอร์ที่ให้บริการ Nitter ครับ

ส่วนตัว ถ้าต้องการใช้งานตามสองข้อข้างบนนี้ (หรือมีเหตุผลอื่น ได้แก่เรื่องความเป็นส่วนตัว เรื่องขี้เกียจดูโฆษณา หรือขี้เกียจดูฟีตแนะนำทวีต) อันนี้เราก็แนะนำ Nitter ครับ
